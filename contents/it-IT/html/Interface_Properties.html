Finestra di dialogo delle proprietà dell'oggetto

Tastiera:
laProprieta
Shortcut:
Fare doppio clic sull'oggetto da modificare.

La finestra di dialogo delle proprietà dell'oggetto è una finestra mobile che consente di modificare le proprietà dell'oggetto di Lands Design selezionato.

Per aprire la finestra di dialogo delle proprietà, eseguire il comando laProprieta e selezionare gli oggetti da modificare. Premere Invio o fare clic con il tasto destro del mouse per terminare la selezione.


Finestra di dialogo relativa alle proprietà dell'oggetto di Lands Design.

La finestra di dialogo delle proprietà dell'oggetto è organizzata in diversi pannelli, in base al tipo di oggetto.

Se è selezionato più di un tipo di oggetto durante l'esecuzione del comando laProprieta, la finestra di dialogo delle proprietà dell'oggetto mostra solamente i pannelli comuni.

Pannelli

  • Generali:
    Tutti gli elementi di Lands Design presentano questo pannello. Stabilisce gli attributi generali dell'oggetto selezionato, come Nome, Colore, Livello, Codice e Parola chiave.
  • Rendering:
    Piante, File di piante, Foreste e Arbusti presentano questo pannello. Stabilisce come le specie di piante vengono mostrate in un file di rendering, in base alla stagione e alla densità del fogliame.
  • Pianta:
    Piante, File di piante, Foreste e Arbusti presentano questo pannello. Stabilisce le proprietà della specie di pianta.
  • Fila:
    Le file di piante e le piante presentano questo pannello. Stabilisce i parametri delle file di piante e delle piante.
  • Coprisuolo:
    Solo gli oggetti coprisuolo presentano questo pannello. Stabilisce i parametri del coprisuolo.
  • Muretto/siepe:
    Solo gli oggetti muretto/siepe presentano questo pannello. Stabilisce i parametri di muretti/siepi.
  • Steccato:
    Solo gli oggetti steccato presentano questo pannello. Stabilisce i parametri dello steccato.
  • Scala:
    Solo gli oggetti scala presentano questo pannello. Stabilisce i parametri della scala.
  • Sentiero:
    Solo gli oggetti sentiero presentano questo pannello. Stabilisce i parametri del sentiero.
  • Terreno:
    Solo i terreni presentano questo pannello. Stabilisce i parametri del terreno.
