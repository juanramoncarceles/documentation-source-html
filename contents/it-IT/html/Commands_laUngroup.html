Separa

Dove posso trovare questo comando?
Tastiera:
laSepara
Icona:
Clic con il tasto sinistro su
Barra degli strumenti:
Menu:
Lands Design > Modifica > Separa
Pannello di modifica:
Selezionare un elemento di Lands, l'area di modifica del pannello di modifica, quindi scegliere Separa.

Il comando laSepara separa boschi, file di piante, file e serie di aspersori, lasciando gli oggetti come entità separate sul disegno tecnico.
