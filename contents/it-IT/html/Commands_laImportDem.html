Importa DEM

Dove posso trovare questo comando?
Tastiera:
laImportaDem
Menu:
Lands Design > Modellazione del terreno > Importa DEM.

Questo comando consente di creare un terreno da un file DEM.

Un file DEM (Digital Elevation Model) è una semplice griglia dei dati di elevazione, con spaziatura regolare.

Dopo aver eseguito il comando laImportaDem, appare la finestra di dialogo di importazione dei file DEM.


Finestra di dialogo di importazione di file DEM.

Questa finestra si divide in due parti. Un'area di navigazione sull'immagine creata da un file DEM e dalle impostazioni di importazione.

Procedimento:

  1. Fare clic sul pulsante "File DEM..." e cercare sul computer un file DEM dal formato compatibile (*.tif, *.tiff, *.png, *.txt o *.asc). Il file verrà importato come immagine in bianco e nero i cui pixel rappresentano valori di elevazione diversi.
  2. Disegnare un rettangolo nella parte superiore dell'immagine, corrispondente all'area da importare.
  3. Specificare le impostazioni di importazione:
    • Curve: importa le curve di contorno della sezione scansionate dall'immagine. È possibile scegliere il livello e il colore di creazione per tali curve. La curve possono essere utilizzate in un secondo momento per creare il terreno con il comando laTerreno.
    • Mesh: viene creata una mesh etichettata come terreno con i punti di elevazione. È possibile scegliere il livello e il colore di creazione per la mesh.
    • Terreno: viene creato automaticamente un terreno con i dati di elevazione scansionati.
  4. Fare clic sul pulsante Importa per importare l'area selezionata.
