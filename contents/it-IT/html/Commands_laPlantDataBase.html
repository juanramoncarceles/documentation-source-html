Lands Design include un esteso database di piante, basato su Visualplant, che può essere usato per inserire determinate specie di piante nel disegno. Può inoltre risultare utile come strumento di riferimento e di consultazione dei dati botanici delle varie specie.

Com'è organizzata questa finestra di dialogo?

La finestra di dialogo del database delle piante è formata dalle seguenti sezioni:


Finestra di dialogo del database delle piante.


  • 1. Sezione relativa alla barra degli strumenti
  • 2. Filtro
  • 3. Elenco delle specie di piante
  • 4. Visualizzazione grafica

Linea della barra degli strumenti.


Barra degli strumenti e finestra di dialogo del database delle piante.

Nuova specie: apre una finestra vuota relativa alle proprietà delle specie di piante, che deve essere compilata con i parametri e caratteristiche della nuova specie vegetale.

Elimina specie: elimina una specie di pianta dall'elenco.

Proprietà: apre la finestra di dialogo relativa alle specie di piante per visualizzare una specie selezionata.

Anteprima scheda: apre una finestra del browser web, che mostra le informazioni sulla specie di pianta selezionata.

Immagini di Google: attiva la ricerca su Google della specie di pianta selezionata e mostra le immagini trovate nell'area di visualizzazione grafica, sostituendo le immagini e illustrazioni esistenti. Per attivare questa opzione, occorre una connessione a Internet. Le immagini trovate vengono archiviate automaticamente nell cartella "Google Plant Images", disponibile nella cartella "Documenti pubblici" di Windows: (C:\Utenti\Pubblica\Documenti pubblici\Lands).

Filtro


Sezione relativa ai filtri del database delle piante.

Questa sezione mostra le varie opzione disponibili per filtrare la ricerca su una specie di pianta.

Per gruppi di specie

Un menu a discesa visualizza tre opzioni che mostrano le specie di piante contrassegnate come preferite , le specie di piante con assegnato un file di rendering specifico (.arPlantX) e tutte le specie di piante.

per nome e nome comune, realizzando una ricerca

Accanto al menu a discesa dei gruppi, c'è una casella di testo di ricerca. Tramite questa casella, digitando il nome o nome comune della pianta desiderata, si visualizzeranno solo le piante corrispondenti alla stringa di testo digitata.

Per caratteristiche delle piante

L'opzione "Filtri" attiva la ricerca delle specie di piante in base alla morfologia e caratteristiche.

Per zone climatiche

Una mappa del mondo e un elenco con varie zone climatiche contraddistinte da un colore. Questa parte dell'area dei filtri può essere nascosta dalla ricerca con un pulsante a comparsa.

Elenco delle specie di piante


Elenco delle specie di piante nel database di piante.

Questa sezione mostra l'elenco delle specie di piante in base alla ricerca. L'elenco può essere ordinato alfabeticamente o numericamente facendo clic sul titolo della categoria corrispondente:

  • Stato di preferito: mostra se una specie è contrassegnata come preferita. Fare clic sull'icona per aggiungere o rimuovere la specie dal gruppo dei preferiti.
    • Preferito
    • Non preferito
  • Nome
  • Nome comune
  • Tipo di pianta
  • Tipo di foglie
  • Altezza massima
  • Forma

Visualizzazione grafica


Area di visualizzazione grafica nel database delle piante.

Quest'area è suddivisa due parti:

  • Parte superiore: mostra alcune immagini o illustrazioni della specie di pianta selezionata, indipendentemente dal fatto che l'opzione "Immagini di Google" sia attiva o meno.
  • Parte inferiore: mostra le rappresentazioni 2D (disegno della corona e configurazione del simbolo) e 3D (rappresentazione concettuale, dettagliata e realistica) delle specie di piante presenti nel modello. Un clic su ciascuna rappresentazione 2D/3D apre la finestra di dialogo relativa alle proprietà delle specie di piante dal pannello Visualizza.

Proprietà delle specie di piante


Finestra di dialogo relativa alle proprietà delle specie di piante, pannello sulle informazioni delle specie


Finestra di dialogo relativa alle proprietà delle specie di piante mostra le informazioni complete su ciascuna specie di pianta, divise in vari pannelli:

Informazioni sulle specie

Questo pannello mostra le informazioni di base sulle specie di piante: origine, famiglia, zone climatiche, requisiti, utilizzo e commenti.

Morfologia

Le seguenti informazioni possono essere specificate per ciascuna specie di pianta.

  • Morfologia: tipo di pianta, tipo di foglia, sistema della radice, crescita e altezza massima.
  • Stagione: periodo di fioritura, colore dei fiori, periodo e colore di fruttificazione.
  • Necessità idriche
  • Descrizione foglie
  • Note dell'utente

Caratteristiche

Le seguenti informazioni possono essere specificate per ciascuna specie di pianta.

  • Caratteristiche: tipo di pianta, tipo di foglia, forma, sistema della radice, crescita e altezza massima.
  • Tipo di suolo
  • Livelli di resistenza all'inquinamento, all'ambiente marino, alla saturazione idrica, al vento, alle infestazioni e al sole.
  • Descrizione foglie
  • Note dell'utente

Rappresentazione

Questo pannello mostre la rappresentazione 2D e 3D per i disegni 2D delle specie di piante, la forma concettuale 3D e la forma 3D dettagliata.
Disegno della corona 2D Forma concettuale 3D Forma dettagliata 3D

Rendering

Questo pannello mostra il file di rendering della pianta (file .ArPlantX), assegnato a questa specie di pianta. Fare clic sul pulsante Modifica per cercare un nuovo file .ArPlantX nella libreria di Lands Design o fare clic sul pulsante relativo all' editor delle piante per aprirlo e modificare il file nXt attuale.
