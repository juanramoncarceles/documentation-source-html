Actualizar

¿Dónde puedo encontrar este comando?
Teclado:
laActualiza
Icono:
clic izquierdo
Barra de herramientas:
Menú:
Lands Design > Edición > Actualizar

Cuando se realizan cambios que no se ven reflejados en el modelo automáticamente, basta con seleccionar el/los objeto/s editado/s y pulsar el icono Actualizar.

NOTA: el proceso de actualización automática ralentiza el rendimiento del software. Por este motivo, solo algunos comandos ofrecen esta opción.
