Supersombreado

¿Dónde puedo encontrar este comando?
Teclado:
laSupersombreado
Menú:
Lands Design > Documentación > Supersombreado

El comando Supersombreado crea una entidad de superficie texturizada basada en una repetición de bitmap.

  • Insertar un supersombreado
  • Puntos de control de supersombreado
  • Opciones y parámetros de inserción
  • Representación gráfica

Pasos:

  1. Ejecute el comando laSupersombreado. Seleccione una de las dos opciones de la línea de comandos para insertar el objeto Supersombreado:
    • Designe un punto interno: haga clic en un contorno cerrado de curvas 2D.
    • Límites: seleccione una curva cerrada y plana.
  2. Después del primer paso, aparecerá el cuadro de diálogo de inserción de supersombreado. Este diálogo muestra las opciones básicas del objeto Supersombreado en una sola ficha:
    • Supersombreado: parámetros de supersombreado.
    Elija las opciones de inserción básicas. Pulse Intro, Esc o haga clic con el botón derecho para cerrar el diálogo.

Puntos de control

El objeto Supersombreado tiene los mismos puntos de control que las curvas utilizadas para generarlo o la curva de contorno generada a partir de los límites de contorno del punto seleccionado al insertarlo.

Además, tiene una flecha de control para mover toda la superficie.

Para activar los puntos de control de un Supersombreado, ejecute el comando ActivarPuntos (o pulse F10) y seleccione el Supersombreado. Para desactivar los puntos de control, pulse Esc o ejecute el comando DesactivarPuntos.

Opciones y parámetros de inserción

Los parámetros y las opciones de inserción de Supersombreado se dividen en varias fichas, que están disponibles desde diferentes diálogos:

  • Cuadro de diálogo de inserción de Supersombreado (solo disponible al insertar nuevos supersombreados).
  • Sección de propiedades de Supersombreado, en el Panel de edición de Lands Design.
  • Diálogo de propiedades de objeto de Supersombreado.

General

Atributos generales del objeto Supersombreado: Nombre, Color, Capa, Código y Palabras clave.

El objeto Supersombreado se representa como una superficie texturizada que se puede ver en cualquier modo de visualización.
