Desplegar

¿Dónde puedo encontrar este comando?
Teclado:
laDespliega
Menú:
Lands Design > Utilidades > Generar documento para entregar

El comando laDespliega guarda el archivo actual con toda la geometría pero sin ningún dato de Lands Design. Cuando vuelva a abrir el archivo, los elementos de Lands previos ya no se podrán editar como elementos de Lands.
