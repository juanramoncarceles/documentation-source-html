Editar

¿Dónde puedo encontrar este comando?
Teclado:
laEdita
Icono:
clic izquierdo [clic largo]
Barra de herramientas:
Menú:
Lands Design > Edición > Editar
Acceso directo:
Haga doble clic en el objeto para editarlo

El comando laEdita abre el diálogo Explorador de propiedades para seleccionar objetos de Lands Design, y permite editarlos. El diálogo es similar al cuadro de diálogo Propiedades, pero muestra la lista de los objetos seleccionados.

Si no hay objetos seleccionados, el diálogo Propiedades muestra las Propiedades de documento.

Cuando se selecciona más de un objeto de Lands Design, el diálogo de Propiedades muestra la lista de las entidades seleccionadas, agrupadas por tipo de elemento (planta, terreno, bosque, arbusto, etc.).
