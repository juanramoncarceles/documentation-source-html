Renderizado

Los objetos de Lands Design admiten materiales de todos los motores de renderizado compatibles con Rhino. La asignación de materiales de objeto funciona igual que en Rhino.

Lands Design incluye la tecnología núcleo de nXt del plug-in de renderizado Flamingo nXt. Esto permite a Lands Design generar plantas muy realistas para el renderizado y plantas texturizadas con pocos polígonos para la representación en tiempo real con cualquier motor de renderizado disponible para Rhino, como V-Ray, Brazil, Maxwell, etc.

Se instalará por defecto una amplia librería de plantas de Flamingo nXt asociada a la base de datos de especies de Lands. Estos archivos contienen todas las texturas y materiales para cada especie de planta.

Aunque es recomendable, no se requiere que Flamingo nXt utilice el generador de plantas de nXt y los archivos de nXt asociados a las especies de plantas.

Plantas y parterres realistas

Flamingo nXt destaca sobre otros plug-ins de renderizado por su capacidad para renderizar plantas tridimensionales muy realistas simplemente utilizando una serie de valores para los parámetros que determinan la apariencia de la planta, por ejemplo, la altura de la planta, el arco y la densidad de las ramas, el grosor, la sinuosidad del tronco, etc. Estos valores se guardan como archivos de definición de plantas.

Lands Design permite al usuario utilizar las plantas nXt realistas asignando un archivo de definición de planta a las Especies de plantas y Parterres.

Cuando se asigna un archivo de nXt, se incrementan las propiedades de las plantas y los parterres de Lands Design. Las nuevas propiedades no tienen efecto en la apariencia de las plantas en el dibujo y solo se utilizarán al renderizar con nXt.

En el Panel de edición están disponibles las siguientes propiedades de renderizado:

  • Estación: cuando se activa la casilla Ignorar estación global, el usuario puede elegir en qué estación se renderizarán las plantas de entre las 7 opciones disponibles: Inicio de primavera, Primavera, Final de primavera, Verano, Final de verano, Otoño e Invierno.
  • Densidad de follaje: Flamingo nXt generará más o menos hojas según el nivel elegido.

Lista de productos de animación y renderizado relacionados con Rhino:

  • Air Rendering Software: renderizado fotorrealista, ilustración, IpR (plug-in)
  • Bongo: animación para diseñadores (plug-in)
  • Brazil para Rhino: herramientas de renderizado para profesionales (plug-in)
  • Flamingo nXt: renderizado por trazado del rayos y radiosidad (plug-in)
  • Hayabusa 2.0 for Rhino: renderizador de alta calidad en tiempo real para Rhinoceros (complemento compatible con 3DM)
  • KeyShot: aplicación de iluminación con trazado de rayos en tiempo real (complemento compatible con 3DM)
  • Maxwell Render, simulador de luz: plug-in Maxwell Render para Rhino (plug-in)
  • NuGraf for large CAD rendering: trazado de rayos multiproceso y conversión 3D para Rhino (plug-in)
  • Penguin: renderizador conceptual no fotorrealístico (plug-in)
  • Renderosity: foro en línea de Rhino (enlace útil)
  • RhinoMan: plug-in de renderizado compatible con RenderMan (plug-in)
  • V-Ray para Rhino: renderizador de iluminación global avanzada (plug-in)
  • VSR Real-time Renderer: renderizador en tiempo real totalmente integrado, disponible como modo de visualización de Rhino (plug-in)
