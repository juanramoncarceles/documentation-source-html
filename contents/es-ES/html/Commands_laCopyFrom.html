Copiar propiedades de otro objeto

¿Dónde puedo encontrar este comando?
Teclado:
laCopiarDesde
Menú:
Lands Design > Edición > Copiar propiedades de otro objeto
Panel de edición:
Seleccione un elemento de Lands > Área de edición del Panel de edición > Ajustar > Copiar propiedades de otro objeto

El comando laCopiaDe cambia las propiedades del objeto de Lands seleccionado para igualar las de otro objeto de Lands.

Este comando está disponible desde el área de edición del Panel de edición para los siguientes objetos:

  • Filas de plantas
  • Bosque
  • Arbusto
  • Murete/seto
  • Valla
  • Fila
  • Camino
  • Escalera
  • Zona

Pasos:

  1. Seleccione el objeto que desee editar.
  2. Seleccione el objeto cuyos parámetros quiere copiar.
