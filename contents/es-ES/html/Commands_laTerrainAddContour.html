Añadir curva de nivel de terreno

¿Dónde puedo encontrar este comando?
Teclado:
laTerrenoNuevoNivel
Icono:
clic izquierdo
Barra de herramientas: Terreno de Lands
Menú:
Lands Design > Modelado de terreno > Terreno > Añadir curva de nivel
Panel de edición:
Seleccione un terreno > Área planta del Panel de edición > Terreno > Añadir curva de nivel

Este comando añade una nueva entrada de datos a un terreno existente. Estos datos de entrada pueden ser curvas 3D, curvas de nivel o nubes de puntos.

Pasos:

  1. Designe un terreno.
  2. Seleccione los nuevos datos de entrada de terreno (curvas 3D, curvas de nivel o nubes de puntos) para añadir al terreno.


Terreno existente con la curva para añadir como nueva línea de curva de nivel.


Terreno después de añadir una nueva curva de nivel.

Las nuevas curvas de nivel añadidas (como otras curvas de nivel o datos de entrada) se pueden quitar de la lista de datos de elevación, en el área de edición de terreno del Panel de edición y el diálogo Propiedades, en la ficha Datos de entrada, después de seleccionar el terreno.
