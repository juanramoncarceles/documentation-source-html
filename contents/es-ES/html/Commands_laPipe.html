Insertar una tubería

Cuando ejecute el comando laTubería, aparecerá el cuadro de diálogo de inserción de tuberías. Este diálogo muestra las opciones básicas en una sola ficha.

  • Tubería: parámetros de tubería.

Pasos:

  1. Seleccione los parámetros de Tubería. Pulse Aceptar para cerrar el diálogo.
  2. Seleccione una de las dos opciones de la línea de comandos:
    • Nueva
    • Existente
  3. Pulse Intro, Esc o haga clic con el botón derecho para finalizar el comando.


Cuadro de inserción del objeto Tubería.



Puntos de control

El objeto Tubería tienen el mismo número de puntos de control que las curvas utilizadas para generarlo.

Para activar los puntos de control de una tubería, ejecute el comando ActivarPuntos (o pulse F10) y seleccione la tubería. Para desactivar los puntos de control, pulse Esc o ejecute el comando DesactivarPuntos.

Opciones y parámetros de inserción

Los parámetros y las opciones de inserción de Tubería se dividen en varias fichas, que están disponibles desde diferentes diálogos:

  • Cuadro de diálogo de inserción de Tubería (solo disponible al insertar nuevas tuberías).
  • Sección de propiedades de Tubería, en el Panel de edición de Lands Design.
  • Diálogo de propiedades de objeto de Tubería.

General

Atributos generales del objeto Tubería: Nombre, Color, Capa, Código y Palabras clave.

Tubería

  • Sección (diámetro): cota de diámetro de la tubería.
  • Presión: presión del agua dentro de las tuberías calculada en atm.
  • Material: campo para especificar el material de la tubería.
  • Description: campo para añadir notas sobre las tuberías.

Opciones de edición

Estas son las opciones de edición del objeto de tubería, disponibles en el área de edición del Panel de edición:

  • Actualizar
  • Copiar propiedades de otro objeto.
  • Ajustar
    • laElevaElevación
    • A cota cero
    • A terreno
  • Dejar de ajustar a terrenos

Representación gráfica de tubería

Las tuberías se representan como líneas 2D en los modos de representación 2D y 3D.
