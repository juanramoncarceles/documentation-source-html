Identificar plantas

Teclado:
laEtiquetaPlantasSeleccionaPlantas
Icono:
clic izquierdo
Barra de herramientas: Etiqueta de plantas de Lands
Menú:
Lands Design > Documentación > Etiqueta de plantas > Identificar plantas

El comando laEtiquetaPlantasSeleccionaPlantas permite identificar las plantas vinculadas a un a etiqueta de planta.

Pasos:

  1. Seleccione la Etiqueta de planta cuyas plantas quiere identificar. Haga clic con el botón derecho o pulse Intro cuando termine.

Las plantas vinculadas al objeto de etiqueta quedarán resaltadas.
