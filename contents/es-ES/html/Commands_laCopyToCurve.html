Copiar en otra curva

¿Dónde puedo encontrar este comando?
Teclado:
laCopiaEnCurva
Menú:
Lands Design > Edición > Copiar propiedades a otra curva
Panel de edición:
Seleccione un elemento de Lands > Área de edición del Panel de edición > Copiar en otra curva

El comando laCopiaEnCurva crea nuevos elementos de Lands seleccionando elementos ya existentes y aplicando sus propiedades a nuevas curvas.

Este comando está disponible desde el área de edición del Panel de edición para los siguientes objetos:

  • Filas de plantas
  • Bosque
  • Murete/seto
  • Valla
  • Fila
  • Camino
  • Escalera
  • Zona

Pasos:

  1. Seleccione el elemento de Lands cuyos parámetros quiera aplicar a un nuevo objeto.
  2. Designe una curva existente en el modelo para definir el nuevo objeto de camino o contorno. O bien, seleccione una opción diferente en la línea de comandos para dibujar el nuevo objeto conforme a otras curvas:
    • Polilínea: el objeto se creará como si dibujara una polilínea.
    • Spline: el objeto se creará como si dibujara una spline.
    • Círculo: el objeto se creará como si dibujara un círculo. El primer clic determina el centro del círculo y el segundo clic determina el radio.
    • Arco: el objeto se creará como si estuviera dibujando un arco. El primer clic designa el punto de inserción. El segundo clic determina un punto medio en el arco. El tercer y último clic determina dónde termina el arco.
    • Rectángulo: el objeto se creará como si dibujara un rectángulo. El primer clic determina una esquina del rectángulo y el segundo clic determina la esquina opuesta.
