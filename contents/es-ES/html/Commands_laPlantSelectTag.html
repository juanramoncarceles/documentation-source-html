Identificar etiqueta (Etiquetar)

Teclado:
laPlantaSeleccionaEtiqueta
Icono:
clic izquierdo
Barra de herramientas:
Menú:
Lands Design > Documentación > Etiqueta de plantas > Identificar etiqueta

El comando laPlantaSeleccionaEtiqueta permite identificar el objeto de etiqueta de una selección de especies de plantas.

Pasos:

  1. Seleccione la planta o especies de plantas cuya etiqueta quiera identificar. Haga clic con el botón derecho o pulse Intro cuando termine.

El objeto de etiqueta quedará resaltado.
