Identificar etiqueta

¿Dónde puedo encontrar este comando?
Teclado:
laPlantaSeleccionaEtiqueta
Icono:
clic izquierdo [clic largo]
Barra de herramientas: Etiqueta de plantas de Lands
Menú:
Lands Design > Documentación > Etiqueta de plantas > Identificar etiqueta
Panel de edición:
Seleccione una planta > Área planta del Panel de edición > Planta > Identificar etiqueta

El comando laPlantaSeleccionaEtiqueta identifica la etiqueta que está mostrando la información de una especie de planta y la resalta.

Se requiere una Etiqueta de plantas para identificarla, de lo contrario no se resaltará ningún objeto.

Pasos:

  1. Seleccione las Especies de plantas cuya etiqueta quiera identificar.
