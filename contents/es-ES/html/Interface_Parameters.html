Parámetros

Los parámetros se utilizan para añadir información a los objetos de geometría. Estos objetos pueden ser elementos de Lands Design (plantas, vallas, caminos, terrenos, etc.) o elementos de Rhino (puntos, curvas, superficies, polisuperficies, extrusiones, mallas y bloques).

Los parámetros se pueden crear y editar de dos formas diferentes:

  • Por documento
  • Por objeto individualmente

Los parámetros existentes se pueden ver para cada entidad de objeto en el panel Propiedades de Rhino , debajo del icono Parámetros , desde donde se pueden añadir los valores.


Parámetros en el panel Propiedades

Parámetros por documento

¿Dónde puedo encontrar este comando?
Teclado:
PropiedadesDeDocumento
Menú:
Herramientas > Opciones > Propiedades de documento > Parámetros

Los Parámetros del documento se controlan desde el cuadro de diálogo Opciones de Rhino > Propiedades de documento > Parámetros. Estos parámetros están disponibles para cada objeto de geometría del modelo.

Los parámetros existentes se representan en una lista con su Categoría y Tipo correspondiente.


Sección Parámetros en las Propiedades de documento de Rhino

  • Añadir: abre el diálogo Propiedades de parámetros para crear un nuevo parámetro.
  • Editar: abre el diálogo Propiedades de parámetros para editar un parámetro seleccionado de la lista.
  • Quitar: elimina un parámetro seleccionado de la lista.

La información de los parámetros personalizados agregados por documento a los elementos de Lands Design se puede listar con las herramientas de listado de Lands Design.

Parámetros por objeto

Los parámetros por objeto son los que se gestionan para cada objeto individualmente. Se pueden crear, editar y eliminar desde el panel Propiedades de Rhino , debajo del icono Parámetros después de seleccionar cualquier tipo de objeto.

  • Abre el diálogo Propiedades de parámetros para crear un nuevo parámetro.
  • Abre el diálogo Propiedades de parámetros para editar el parámetro. Debe seleccionar el nombre del parámetro para activar este botón.
  • Elimina un parámetro existente. Debe seleccionar el nombre del parámetro para activar este botón.

Nota: el Nombre y la Descripción son parámetros nativos y no se pueden editar.

Cuadro de diálogo Propiedades de parámetros


Cuadro de diálogo Propiedades de parámetro

  • Nombre: nombre del parámetro.
  • Categoría: sección donde el parámetro será visible. Puede intruducir una nueva categoría o seleccionar una de las categorías existentes (General, Visualización, Geometría y Posición), además de las nuevas categorías que se hayan creado.
  • Tipo de dato: indica el formato del valor. Seleccione una de las opciones de la lista.
  • Descripción: campo para añadir notas sobre el parámetro.
