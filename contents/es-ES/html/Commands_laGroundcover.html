Los parterres de Lands Design son áreas de vegetación llenas de especies de plantas según parámetros de densidad y se representan con un patrón de sombreado para la visualización en 2D.

Insertar un parterre:

Cuando ejecute el comando laParterre, aparecerá el cuadro de diálogo de inserción de parterre. Este diálogo muestra las opciones básicas y los parámetros del objeto Parterre en una sola ficha.

Pasos:

  1. Seleccione una especie de parterre de la lista. Haga clic en el botón Examinar... para seleccionar otra desde la Base de datos de plantas.
  2. Seleccione el resto de parámetros de inserción (modo de presentación, patrón de sombreado y archivo de planta de nXT) y pulse Aceptar para cerrar el diálogo.
  3. Designe una curva existente en el modelo para definir un contorno de parterre, o seleccione una opción diferente en la línea de comandos.
    • Polilínea: el parterre se creará como si dibujara una polilínea.
    • Spline: el parterre se creará como si dibujara una spline.
    • Círculo: el parterre se creará como si dibujara un círculo. El primer clic determina el centro del círculo y el segundo clic determina el radio.
    • Arco: el parterre se creará como si estuviera dibujando un arco. El primero clic determina el punto de inserción. El segundo determina el clic determina un punto medio en el arco. El tercer clic determina dónde termina el arco.
    • Rectángulo: el parterre se creará como si dibujara un rectángulo. El primer clic determina una esquina del rectángulo y el segundo clic determina la esquina opuesta.
  4. Pulse Intro, Esc o haga clic con el botón derecho para finalizar el comando.


Cuadro de inserción del objeto Parterre.



Puntos de control

Los parterres tienen el mismo número de puntos de control que las curvas utilizadas para generarlos.

Además, tienen una flecha de control de movimiento para mover todo el objeto de parterre.

Los puntos de control de los parterres se activan automáticamente al seleccionar los parterres uno por uno. Para activar uno o más puntos de control del objeto Parterre, seleccione los parterres y ejecute el comando ActivarPuntos (o pulse F10). Para desactivar los puntos de control, pulse Esc o ejecute el comando DesactivarPuntos.

Opciones y parámetros de inserción

Los parámetros y las opciones de inserción de Parterre se dividen en varias fichas, que están disponibles desde diferentes diálogos:

  • Cuadro de diálogo de inserción de Planta (solo disponible al insertar nuevas plantas).
  • Sección de propiedades de Planta, en el Panel de edición de Lands Design.
  • Diálogo de propiedades de objeto Planta.

General

Atributos generales del objeto Parterre: Nombre, Color, Código y Palabras clave.

Cubresuelos

  • Especie: nombre del parterre o de la especie de planta seleccionada. El cuadro combinado muestra una lista de las diferentes especies de vegetación que ya se han insertado en el modelo actual. Puede seleccionar una especie de planta de la lista o hacer clic en el botón Examinar... para seleccionar una nueva especie de la Base de datos de plantas.
  • Modo de presentación: hay tres tipos de modos de presentación disponibles, pero se pueden crear nuevos.
    • Raíz desnuda
    • Cepellón
    • Contenedor
  • Densidad: indica la cantidad de muestras de vegetación en unidades/m2 dentro de la curva de contorno.
  • Patrón de sombreado: parámetros para la representación 2D del parterre.
  • Archivo de planta de nXT: los parterres usan archivos de parterre (".ArGroundcoverX") para sus representaciones en 3D.

Opciones de edición

Estas son las opciones de edición del objeto de parterre, disponibles en el área de edición del Panel de edición:

  • Actualizar
  • Copiar propiedades de otro objeto
  • Aleatoriza
  • Extraer curvas
  • Ajustar
    • laElevaElevación
    • A cota cero
    • A terreno
  • Dejar de ajustar a terrenos
  • Parterre
    • Etiquetar (este comando aparece solamente cuando el parterre no tiene ninguna etiqueta asignada)
    • Identificar etiqueta (este comando aparece solamente cuando hay una etiqueta de plantas que hace referencia al parterre en el modelo)
    • Insertar foto
    • Añadir hueco: este comando sustrae un contorno de un parterre.

Representación gráfica de parterre:

Lands Design ofrece diferentes representaciones simultáneas en 2D y 3D de los parterres en el dibujo, con el fin de manejar las plantas desde tres niveles conceptuales distintos. La visualización de la representación del Parterre se puede cambiar en Propiedades de documento.
