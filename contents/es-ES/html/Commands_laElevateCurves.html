Elevar curvas

¿Dónde puedo encontrar este comando?
Teclado:
laElevaCurvas
Icono:
clic izquierdo
Barra de herramientas: Terreno de Lands:
Menú:
Lands Design > Modelado de terreno > Elevar curvas

Este comando eleva las curvas cada cierta distancia para utilizarlas para crear un terreno.

Cuando ejecute el comando laElevaCurvas, aparecerá el cuadro de diálogo Elevar curvas.

Pasos:

  1. Defina los parámetros de Elevar curvas en el cuadro de diálogo:
    • Cota inicial: elevación de la primera curva a elevar.
    • Incremento. Distancia incremental entre las curvas de nivel a elevar.
  2. Haga clic en el botón Trazar línea y trace una línea en el modelo que se cruce con las curvas de nivel a elevar.
  3. Haga clic en la casilla de verificación Invertir para invertir el orden de elevación de las curvas.
  4. Haga clic en el botón Aplicar y cierre el diálogo.
