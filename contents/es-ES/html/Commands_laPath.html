Insertar un camino:

Cuando ejecute el comando laCamino, aparecerá el cuadro de diálogo de inserción de caminos. Este diálogo muestra las opciones básicas del objeto Camino en dos fichas.

  • Camino: parámetros de camino.
  • Material: opciones de material para el objeto Camino.

Pasos:

  1. Seleccione el perfil de sección del camino. Ya se ha asignado un perfil rectangular (de 3 x 0,2 metros) de forma predeterminada. También puede seleccionar una curva existente en el modelo (opción Seleccionar) o dibujarla en el modelo (opción Designar punto a punto).
  2. Seleccione el resto de parámetros de Camino. Pulse Aceptar para cerrar diálogo.
  3. Designe una curva existente en el modelo para crear el camino, o seleccione una opción diferente en la línea de comandos.
    • Polilínea: el camino se creará como si dibujara una polilínea.
    • Spline: el camino se creará como si dibujara una spline.
    • Círculo: el camino se creará como si dibujara un círculo. El primer clic determina el centro del círculo y el segundo clic determina el radio.
    • Arco: el camino se creará como si estuviera dibujando un arco. El primero clic determina el punto de inserción. El segundo clic determina un punto medio en el arco. El tercer y último clic determina dónde termina el arco.
    • Rectángulo: el camino se creará como si dibujara un rectángulo. El primer clic determina una esquina del rectángulo y el segundo clic determina la esquina opuesta.
  4. Pulse Intro, Esc o haga clic con el botón derecho para finalizar el comando.

Nota: asegúrese de que la curva seleccionada como perfil de transversal del camino esté alineada con el eje XY.

Puntos de control

Los objetos de Camino tienen tres tipos diferentes de puntos de control:

  1. Puntos de control del eje del camino: son los mismos puntos de control que los de las curvas utilizadas para generar los caminos.
  2. Flecha de control de movimiento: flecha situada al principio del objeto camino, que se utiliza para mover todo el objeto.
  3. Puntos de control del perfil transversal: estos puntos de control se usan para modificar el perfil transversal del camino. Son los mismos puntos de control que los de las curvas utilizadas para definir el perfil de sección del camino. Se encuentran al inicio del objeto de camino.

Los puntos de control de los caminos se activan automáticamente al seleccionar los caminos uno por uno. Para activar uno o más puntos de control del objeto Camino, seleccione los caminos y ejecute el comando ActivarPuntos (o pulse F10). Para desactivar los puntos de control, pulse Esc o ejecute el comando DesactivarPuntos.

Opciones y parámetros de inserción

Los parámetros y las opciones de inserción de Camino se dividen en varias fichas, que están disponibles desde diferentes diálogos:

  • Cuadro de diálogo de inserción de Camino (solo disponible al insertar nuevos caminos).
  • Sección de propiedades de Camino, en el Panel de edición de Lands Design.
  • Diálogo de propiedades de objeto de Camino.

General

Atributos generales del objeto Camino: Nombre, Color, Capa, Código y Palabras clave.

Camino

  • Perfil de sección: curva usada para definir el perfil de sección del Camino.
  • Suavidad: precisión en la que el objeto Camino se subdivide a lo largo de la curva del eje para ajustarse a su trayectoria. Este parámetro solo es relevante cuando el objecto Camino se ha creado desde una curva curvada.
  • Justificación: marca la posición del perfil de sección del camino desde el que se extruirá a lo largo de la curva de eje del camino. Hay nueve justificaciones posibles. La justificación Ninguna mantiene la alineación de la trayectoria a lo largo del eje del camino, aunque modifique el perfil de sección del camino.

Material

Esta ficha permite cambiar la imagen texturizada asignada a este elemento y definir el tamaño de la imagen y la reflectividad.

Opciones de edición

Estas son las opciones de edición del objeto de camino, disponibles en el área de edición del Panel de edición:

  • Actualizar
  • Convertir a 2D. Muestra el objeto en representación 2D.
  • Convertir a 3D. Muestra el objeto en representación 3D.
  • Mostrar 2D y 3D: Activa la representación 2D y 3D del objeto simultáneamente.
  • Copiar propiedades de otro objeto
  • Copiar en otra curva
  • Invertir curva
  • Extraer curva
  • Ajustar
    • laElevaElevación
    • A cota cero
    • A terreno
  • Dejar de ajustar a terrenos
  • Camino
    • Editar perfil de sección: este comando coloca un plano de construcción alineado con el perfil de sección del camino al inicio del objeto. Facilita la edición del perfil del camino a través de sus puntos de control. Ejecute el comando PlanoC (y seleccione Universal > Superior) en la línea de comandos para restablecer la posición del plano construcción.
