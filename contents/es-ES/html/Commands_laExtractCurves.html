Extraer curvas

¿Dónde puedo encontrar este comando?
Teclado:
laExtraeCurvas
Menú:
Lands Design > Edición > Extraer curvas
Panel de edición:
Seleccione un elemento de Lands > Área de edición del Panel de edición > Extraer curvas

El comando laExtraeCurvas obtiene las curvas usadas para generar los elementos de Lands.

Este comando está disponible desde el área de edición del Panel de edición para los siguientes objetos:

  • Filas de plantas
  • Bosque
  • Arbusto
  • Murete/seto
  • Valla
  • Fila
  • Camino
  • Escalera
  • Zona

Pasos:

  1. Seleccione el objeto desde el que extraer las curvas.
