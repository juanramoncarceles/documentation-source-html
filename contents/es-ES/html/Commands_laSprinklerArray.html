La matriz del aspersor inserta una matriz de aspersores que cubre un área específica. El área se define mediante una curva de cerrada, que puede ser de cualquier tipo: polilínea, spline, círculo, etc.

Insertar una matriz de aspersores

Cuando ejecute el comando laMatrizAspersores, aparecerá el cuadro de diálogo de inserción de matriz de aspersores. Este diálogo muestra las opciones básicas en dos fichas diferentes.

  • Aspersor
  • Matriz

Pasos:

  1. Seleccione los parámetros del Aspersor desde la ficha Aspersor y los parámetros de Matriz desde la pestaña Matriz. Pulse Aceptar para cerrar el diálogo.
  2. Designe una curva existente en el modelo para definir el contorno de la matriz de aspersores, o seleccione una opción diferente en la línea de comandos.
    • Polilínea: la matriz de aspersores se creará como si dibujara una polilínea.
    • Spline: la matriz de aspersores se creará como si dibujara una spline.
    • Círculo: la matriz de aspersores se creará como si dibujara un círculo. El primer clic determina el centro del círculo y el segundo clic determina el radio.
    • Arco: la matriz de aspersores se creará como si estuviera dibujando un arco. El primero clic determina el punto de inserción. El segundo determina el clic determina un punto medio en el arco. El tercer clic determina dónde termina el arco.
    • Rectángulo: la matriz de aspersores se creará como si dibujara un rectángulo. El primer clic determina una esquina del rectángulo y el segundo clic determina la esquina opuesta.
  3. Pulse Intro, Esc o haga clic con el botón derecho para finalizar el comando.


Cuadro de inserción del objeto Matriz de aspersores.



Nota: la matriz de aspersores se crea como objeto agrupado. Sin embargo, puede desagruparla con el comando Desagrupar. En este caso, obtendrá entidades individuales de aspersores y se perderán las funciones de matriz.

Puntos de control

Las matrices de aspersores tienen el mismo número de puntos de control que las curva utilizadas para generarlas.

Además, tiene una flecha de control de movimiento para mover todo el objeto.

Para activar los puntos de control de la matriz de aspersores, ejecute el comando ActivarPuntos (o pulse F10) y seleccione la matriz de aspersores. Para desactivar los puntos de control, pulse Esc o ejecute el comando DesactivarPuntos.

Opciones y parámetros de inserción

Los parámetros y las opciones de inserción de Matriz de aspersores se dividen en varias fichas, que están disponibles desde diferentes diálogos:

  • Cuadro de diálogo de inserción de Matriz de aspersores (solo disponible al insertar nuevas matrices de aspersores).
  • Sección de propiedades de Matriz de aspersores, en el Panel de edición de Lands Design.
  • Diálogo de propiedades de objeto de Matriz de aspersores.

General

Atributos generales del objeto Matriz de aspersores: Nombre, Color, Capa, Código y Palabras clave.

Aspersor

  • Tipo: existen cuatro tipos de aspersores:
    • Aspersor
    • Difusor
    • Turbina de agua
    • Boca de riego
  • Description: campo para añadir notas sobre los aspersores.
  • Símbolo: dibujo 2D utilizado para representar las unidades de aspersores dentro del área de la matriz. Haga clic en Examinar... para cargar un bloque 2D desde la librería de Lands Design o desde cualquier otro directorio.
  • Radio
  • Ángulo de barrido

Matriz

  • Distribución: los aspersores se pueden distribuir de forma de matriz cuadrada o tresbolillo.
  • Distancia entre aspersores: distancia expresada en forma de porcentaje del diámetro del aspersor.

Opciones de edición

Estas son las opciones de edición del objeto de matriz de aspersores, disponibles en el área de edición del Panel de edición:

  • Actualizar
  • Copiar propiedades de otro objeto.
  • Ajustar
    • laElevaElevación
    • A cota cero
    • A terreno
  • Dejar de ajustar a terrenos
  • Matriz de aspersores
    • Añadir hueco: comando para sustraer un contorno del área de una matriz de aspersores

Representación gráfica de la matriz de aspersores:

El objeto Matriz de aspersores ofrece una única representación en 2D simultánea en el dibujo.
