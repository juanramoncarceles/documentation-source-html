Marcar objeto como planta

¿Dónde puedo encontrar este comando?
Teclado:
laMarcaObjetoComoPlanta
Menú:
Lands Design > Vegetación > Marcar objeto como planta
Panel de edición:
Seleccione un objeto de Rhino > Área de edición del Panel de edición > Marcar objeto como planta

El comando laMarcaObjetoComoPlanta convierte cualquier pieza de geometría en una planta de Lands, para que se le puedan asignar propiedades de especies de plantas y se pueda listar en el listado de plantas.

Este comando está disponible para cualquier objeto, excepto los elementos de Lands.

Cuando ejecute el comando MarcaObjetoComoPlanta, aparecerá el cuadro de diálogo de inserción para marcar objetos como plantas. Este diálogo muestra las opciones básicas de la nueva planta en una ficha.

Pasos:

  1. Seleccione una especie de planta de la lista. Haga clic en el botón Examinar... para seleccionar otra desde la Base de datos de plantas.
  2. Seleccione el resto de parámetros de inserción (Calibre, Modo de presentación, Cantidad y Densidad) y haga clic en Aceptar para cerrar el cuadro de diálogo.


Cuadro de diálogo Marcar objeto como planta.


Puede deshacer esta operación ejecutando el comando laDesmarcaObjetoComoPlanta disponible en el Área de edición en el Panel de edición después de seleccionar el objeto.
