Auf Gelände erhöhen

Wo befindet sich dieser Befehl?
Tastatur:
laAufGeländeErhöhen
Symbol:
Linksklick
Werkzeugleiste:
Menü:
Lands Design > Bearbeiten > Anpassen > An Gelände
Bearbeitungspanel:
Ein Lands-Element auswählen > Bearbeitungsbereich im Bearbeitungspanel > Anpassen > Auf Gelände

Mit dem Befehl laAufGeländeErhöhen werden Objekte jeglicher Art vertikal verschoben, bis sie auf der Fläche eines Lands-Design-Geländes zu liegen kommen.

Wenn das zu verschiebende Objekt aus verschiedenen Elementen besteht, etwa einem Wald oder einer Reihe, wird jedes Element auf die Geländefläche projiziert.

Vorgehen:

  1. Wählen Sie das an das Gelände anzupassende Objekt aus.

Das ausgewählte Objekt erkennt die Geländefläche und wird darauf projiziert. Das Objekt bleibt auf die Geländefläche aufgesetzt. Wird das Geländeprofil geändert, aktualisiert das Objekt automatisch seine Position.

HINWEIS: wenn das Gelände außerhalb der Geländebegrenzung platziert wird, verbleibt es auf der gleichen Höhenposition, bis es darunter einen Geländebereich findet.
