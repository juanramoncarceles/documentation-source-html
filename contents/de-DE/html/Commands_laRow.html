Reihe einfügen:

Durch Ausführen des Befehls laReihe wird der Einfügedialog für Reihe geöffnet. Dieser Dialog zeigt die grundlegenden Optionen des Reihenobjekts auf zwei verschiedenen Reitern an:

  • Muster: Reihenelement-Parameter.
  • Reihe: Reihenparameter.

Vorgehen:

  1. Wählen Sie ein Reihenelement unter dem Reiter Muster. Dieses Element kann auf drei verschiedene Arten definiert werden:
    • Ein Block aus der Blockbibliothek.
    • Eine Datei aus Ihrem PC.
    • Ein im Modell vorhandenes Objekt.
    Nach der Auswahl wird eine Vorschau des Objekts im Dialog erscheinen.
  2. Wählen Sie die Reihenparameter unter dem Reiter Reihe. Klicken Sie auf OK, um das Dialogfeld zu schließen.
  3. Wählen Sie eine im Modell vorhandene Kurve aus, um die Reihe zu definieren, oder wählen Sie eine unterschiedliche Option in der Befehlszeile.
    • Polylinie: die Reihe wird so erzeugt, als ob Sie eine Polylinie zeichnen würden.
    • Spline: die Reihe wird so erzeugt, als ob Sie einen Spline zeichnen würden.
    • Kreis: die Reihe wird so erzeugt, als ob Sie einen Kreis zeichnen würden. Mit dem ersten Klick wird die Kreismitte bestimmt, und mit dem zweiten Klick der Radius.
    • Bogen: die Reihe wird so erzeugt, als ob Sie einen Bogen zeichnen würden. Mit dem ersten Klick wird der Einfügungspunkt bestimmt. Der zweite Klick bestimmt einen Mittelpunkt im Bogen. Mit dem dritten und letzten Klick wird bestimmt, wo der Bogen endet.
    • Rechteck: die Reihe wird so erzeugt, als ob Sie ein Rechteck zeichnen würden. Mit dem ersten Klick wird eine Ecke des Rechtecks bestimmt, und mit dem zweiten Klick die gegenüberliegende Ecke.
  4. Beenden Sie den Befehl durch Drücken von Eingabe, Esc oder Klicken auf die rechte Maustaste.

Die Reihenparameter können während ihrer Einfügung und nach Beendigung des Befehls bearbeitet werden, und zwar im Lands-Design-Bearbeitungspanel, im Abschnitt Eigenschaften, unter dem Reiter Reihe.


Einfügedialog für das Reihenobjekt.


Hinweis: Als Reihenmuster dient alles außer einer Pflanze. Um Pflanzenreihen zu erzeugen, verwenden Sie den Befehl laPflanzenreihe.


Kontrollpunkte

Reihen haben dieselben Kontrollpunkte wie die zu ihrer Erzeugung verwendeten Kurven, ohne Rücksicht auf die Zahl der Elemente oder ihre Position.

Zuzüglich haben sie einen Kontrollpfeil zum Verschieben des gesamten Objekts.

Reihen-Kontrollpunkte werden bei der Einzelauswahl von Reihen automatisch aktiviert. Um einen oder mehrere Reihen-Kontrollpunkte zu aktivieren, wählen Sie diese einfach aus und starten Sie den Befehl PunkteEin (oder drücken Sie F10). Um die Punkte zu deaktivieren, drücken Sie ESC oder starten Sie den Befehl PunkteAus.

Einfügeoptionen und -parameter

Die Reihen-Einfügungsoptionen und -Parameter sind in unterschiedliche Reiter aufgeteilt und über verschiedene Dialoge abrufbar:

  • Reihen-Einfügungsdialog (nur beim Einfügen neuer Reihen verfügbar).
  • Abschnitt der Reihen-Eigenschaften, im Lands-Design-Bearbeitungspanel.
  • Reihen-Objekteigenschaftendialog.

Allgemein

Allgemeine Attribute für das Reihenobjekt: Name, Farbe, Ebene, Code und Stichwörter.

Muster

Parameter für die Reihenelemente:

  • Muster: Reihenelement. Sie können es entweder aus der Blockbibliothek auswählen oder von Ihrem PC, oder Sie können auf ein im Modell vorhandenes Objekt klicken.
  • Drehung: Drehwinkel jedes Elements vom eigenen Einfügungspunkt.
  • Orientierung: vier Optionen sind verfügbar, um das Reihenelement in hinsichtlich der Reihenachsenkurve auszurichten:
    • Tangential zur Achse
    • Rechtwinklig zur Achse
    • An festem Punkt: Klicken Sie auf die Schaltfläche Punkt wählen, um einen Punkt im Modell auszuwählen.
    • Gegebener Winkel
    Mit dem Kontrollkästchen Umkehren wird die Ausrichtung jedes Elements um 180 Grad gedreht. Diese Option ist nur bei Auswahl der Ausrichtungsoptionen Tangential zur Achse und Rechtwinklig zur Achse verfügbar.

Wenn Reihen mehr als ein Muster enthalten, sind die Parameter jeder einzelnen davon verfügbar, und zwar nach Auswahl des gewünschten Musters von der Objektauswahl-Liste im Lands-Bearbeitungspanel.

Reihe

Zum Verteilen der Elemente in Reihen sind zwei Optionen verfügbar:

  • Anzahl Einheiten: zur Bestimmung der Gesamtmenge von Reihenelementen entlang der Reihen-Achsenkurve.
  • Trennung: zur Platzierung der Reihenelemente in einem gewissen Abstand, gemäß einer Ausrichtung auf der Reihenachsenkurve (am Anfang, zentriert oder am Ende).
  • Abstand zur Achse: zum Verschieben der Position von Elementen über die Pfadkurve.

Reihen können unterschiedliche Elementmuster kombinieren:

  • Anzahl Elemente: Geben Sie die Anzahl verschiedener Muster ein. Die Liste verschiedener Muster wird in der Liste zur Objektauswahl im Lands-Bearbeitungspanel erscheinen, wo sie zur Bearbeitung ihrer Parameter ausgewählt werden können.
  • Verteilung: Optionen zum Verteilen der Reihenelemente, wenn mehr als ein Mustertyp vorhanden ist.

Eingefügte Einheiten: zeigt die genaue Anzahl von Elementeinheiten in der Reihe an.

Material

Mit diesem Reiter können Sie das diesem Element zugewiesene texturierte Bild ändern und Bildgröße und Reflexionsvermögen definieren.

Bearbeitungsoptionen

Dies sind die Bearbeitungsoptionen für das Reihenobjekt, verfügbar im Bearbeitungsbereich des Bearbeitungsfensters:

  • Aktualisieren
  • In 2D umwandeln. Zeigt das Objekt in der 2D-Darstellung an.
  • In 3D umwandeln. Zeigt das Objekt in der 3D-Darstellung an.
  • 2D und 3D anzeigen: Aktiviert gleichzeitig die 2D- und 3D-Darstellung des Objekts.
  • Eigenschaften von einem anderen Objekt kopieren
  • Auf eine andere Kurve kopieren
  • Kurve umkehren
  • Kurve extrahieren
  • Anpassen
    • Erhöhung
    • Auf Nullhöhe
    • An Gelände
  • Nicht an Gelände anpassen

Grafische Darstellung der Reihe:

Das Reihenobjekt bietet eine gleichzeitige 2D- und 3D-Darstellung der Zeichnung, je nach ausgewähltem Element. Wenn das Reihenelement aus der Blockbibliothek ausgewählt wird, bietet es eine 2D- und 3D-Anzeige. Wenn das Element aus dem Modell ausgewählt wird, verfügt es nur über eine 3D-Anzeige.
