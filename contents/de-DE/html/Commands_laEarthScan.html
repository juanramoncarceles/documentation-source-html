Erdscan

Wo befindet sich dieser Befehl?
Tastatur:
laEarthScan
Symbol:
Linksklick
Werkzeugleiste: Lands-Gelände
Menü:
Lands Design > Geländemodellierung > Höhendaten importieren

Mit diesem Befehl wird ein Gelände aus der Cloud gescannt und als texturiertes Polygonnetz oder Geländeobjekt importiert.

Nach Ausführen des Befehls laEarthScan erscheint der Dialog Höhendaten importieren.


Dialog Höhendaten importieren.

Dieser Dialog ist in zwei Teile geteilt. Ein Bereich zum Navigieren über der Erde und die Import-Einstellungen.

Klicken Sie im Navigationsbereich auf die linke obere Schaltfläche, um zwischen 4 verschiedenen Kartenmodi hin- und herzuschalten (Straßen, Außenflächen, Satellit und Satellitenstraßen) oder verwenden Sie das Feld Suchen um einen bestimmten Ort auf der Welt zu finden.

Die Einstellungen können mit der Pfeil-Schaltfläche im unteren Bereich der Einstellungen zusammengefaltet werden.

Vorgehen:

  1. Navigieren Sie über den Bereich der Erde den Sie importieren möchten. Der Bereich hinter den Einstellungen wird ebenfalls mit einbezogen. Verwenden Sie die Pfeile, um die Karte exakt Frame für Frame auf gleicher Höhe zu scrollen. Der Bereich ist auf eine Ausdehnung von 10 Km entlang jeder Seite des Navigationsbereichs beschränkt, ansonsten können die Höhendaten nicht importiert werden.
  2. Wählen Sie beim Import die Geometrieerzeugung aus:
    • Polygonnetz erzeugen: ein texturiertes Polygonnetz, das mit dem Bild des gescannten Bereichs als Gelände etikettiert ist, wird erzeugt.
    • Gelände erzeugen: mit den gescannten Höhedaten wird automatisch ein Gelände erzeugt.
  3. Geolokalisieren Sie das Dokument. Beim erstmaligen Importieren wird die Bildmitte auf den Koordinaten (0,0,0) platziert. Dies stimmt nach Breitengrad, Längengrad und Höhe mit der Modellposition überein. Bei einem zweiten Import können Sie wählen, ob Sie die ursprünglichen Koordinaten des Erstimports beibehalten oder sie mit der neu importierten Kartenmitte überschreiben wollen.
  4. Wählen der Höhendaten: wählen Sie die Genauigkeit der zu scannenden Samples nach einem Werte-Intervall (0-100) oder nach einem bestimmten Abstand zwischen den Samples. Anhand der ausgewählten Werte wird die zum Import benötigte Anzahl Samples berechnet, welche auf 50000 pro Import beschränkt ist.
  5. Klicken Sie auf die Schaltfläche Importieren um die ausgewählten Daten zu importieren. (Klicken Sie auf die Schaltfläche Zurücksetzen um die Originaleinstellungen wiederherzustellen).

HINWEIS 1: Die allgemeine Genauigkeit der aus der Cloud bezogenen Daten beträgt fast ein Sample alle 30 Meter. Wenn Sie in den Einstellungen der Höhendaten einen kleineren Wert festlegen, erzeugt Lands automatisch die interpolierten Samples.

HINWEIS 2: Sobald Sie das Gelände importieren, aktivieren Sie den Anzeigemodus Gerendert, um das Bild auf dem importierten Polygonnetz zu sehen.
