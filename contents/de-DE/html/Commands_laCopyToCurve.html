Auf eine andere Kurve kopieren

Wo befindet sich dieser Befehl?
Tastatur:
laAufKurveKopieren
Menü:
Lands Design > Bearbeiten > Auf eine andere Kurve kopieren
Bearbeitungspanel:
Ein Lands-Element auswählen > Bearbeitungsbereich im Bearbeitungspanel > Auf eine andere Kurve kopieren

Mit dem Befehl laAufKurveKopieren werden neue Lands-Elemente durch Auswahl bereits vorhandener Elemente und Anwendung ihrer Eigenschaften auf neue Kurven erzeugt.

Dieser Befehl ist im Bearbeitungsbereich im Bearbeitungspanel für die folgenden Objekte verfügbar:

  • Pflanzenreihe
  • Wald
  • Teilerwand - Hecke
  • Zaun
  • Reihe
  • Pfad
  • Treppe
  • Zone

Vorgehen:

  1. Wählen Sie die Lands-Elemente aus, deren Parameter Sie auf ein neues Objekt anwenden möchten.
  2. Wählen Sie eine vorhandene Kurve im Modell, um den neuen Objektpfad oder die Begrenzung zu definieren. Oder wählen Sie eine andere Option in der Befehlszeile, um das neue Objekt in Übereinstimmung mit unterschiedlichen Kurven zu zeichnen:
    • Polylinie: das Objekt wird so erzeugt, als ob Sie eine Polylinie zeichnen würden.
    • Spline: das Objekt wird so erzeugt, als ob Sie einen Spline zeichnen würden.
    • Kreis: das Objekt wird so erzeugt, als ob Sie einen Kreis zeichnen würden. Mit dem ersten Klick wird die Kreismitte festgelegt, und mit dem zweiten Klick der Radius.
    • Bogen: das Objekt wird so erzeugt, als ob Sie einen Bogen zeichnen würden. Mit dem ersten Klick wird der Einfügungspunkt festgelegt. Der zweite Klick legt den Mittelpunkt des Bogens fest. Mit dem dritten und letzten Klick wird festgelegt, wo der Bogen endet.
    • Rechteck: das Objekt wird so erzeugt, als ob Sie ein Rechteck zeichnen würden. Mit dem ersten Klick wird eine Ecke des Rechtecks festgelegt, und mit dem zweiten Klick die gegenüberliegende Ecke.
