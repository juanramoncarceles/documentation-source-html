Pflanzenetikett-Befehle


Die Pflanzenetikettbefehle bieten Funktionen zur Etikettierung von Pflanzenarten mit Identifikationscode sowie zur Mengenbestimmung der im Modell enthaltenen Arten.

Pflanzenarten können einzeln oder als verknüpfte Gruppe etikettiert werden.

Pflanzenetikett-Parameter

Allgemein

Allgemeine Attribute für das Pflanzenetikett-Objekt: Name, Farbe, Ebene, Code und Stichwörter.

Etikett

Dieser Reiter definiert das Etikett und das Textformat:

  • Form: Parameter der Art der Informationsanzeige im Etikett und Format der Umrandung.
  • Schriftart: Textattribute und -parameter.
  • Anmerkungspfeil: wenn diese Schaltfläche aktiviert ist, erscheint ein neuer Reiter mit den Anmerkungspfeil-Einstellungen.


Reiter Etikett.


Anmerkungspfeil

Anmerkungspfeil-Einstellungen. Dieser Reiter erscheint nur dann, wenn das Kontrollkästchen Anmerkungspfeil im Reiter Etikett aktiviert ist:

  • Linie: Parameter der Linie des Anmerkungspfeils
  • Ende: Parameter der Spitze des Anmerkungspfeils


Reiter Anmerkungspfeil.
