Verzonen

Wo befindet sich dieser Befehl?
Tastatur:
laVerzonen
Symbol:
Linksklick
Werkzeugleiste: Lands-Dokumentation
Menü:
Lands Design > Dokumentation > Verzonen

Mit dem Befehl Verzonen wird ein Flächenelement erzeugt, das die Information (Name und Quadratmeter) auf einem Bereich des Modells anzeigt.

  • Einfügen einer Zone
  • Zone-Kontrollpunkte
  • Einfügeoptionen und -parameter
  • Bearbeitungsoptionen
  • Graphische Darstellung

Vorgehen:

  1. Führen Sie den Befehl laVerzonen aus. Wählen Sie eine der beiden Optionen in der Befehlszeile aus, um das Zone-Objekt einzufügen:
    • Inneren Punkt: innerhalb einer geschlossenen Begrenzung von 2D-Kurven klicken.
    • Begrenzungskurve: geschlossene und planare Kurve auswählen.
  2. Nach dem ersten Schritt erscheint der Einfügungsdialog für Verzonen. Dieser Dialog zeigt die grundlegenden Optionen des Zone-Objekts auf zwei Reitern an:
    • Zone: Zone-Parameter.
    • Material: Materialeinstellungen für das Zone-Objekt.
    Wählen Sie die grundlegenden Einfügungsoptionen. Schließen Sie den Dialog durch Drücken von Eingabe, Esc oder Klicken auf die rechte Maustaste.
  3. Wählen Sie einen Punkt zum Einfügen des Zone-Etiketts aus.


Einfügedialog für das Verzonungsobjekt.


Eine Zone kann auch beim Hinzufügen einer Teilung auf einem Gelände erstellt werden, welche mit den folgenden Befehlen erzeugt wird:

  • laGeländeUnterteilen
  • laGeländeAuftragUndAbtragHinzufügen
  • laGeländeÖffnungHinzufügen
  • laGeländePfadHinzufügen

Kontrollpunkte

Das Zone-Objekt hat genauso viele Kontrollpunkte wie die Kurven, aus denen es erzeugt wurde.

Zusätzlich hat es einen Kontrollpfeil zum Verschieben der Zonenfläche sowie einen Kontrollpfeil zum Verschieben des Zone-Etiketts.

Um einen Zonenobjekt-Kontrollpunkt zu aktivieren, führen Sie den Befehl PunkteEin aus (oder drücken Sie F10) und wählen Sie das Zone-Objekt. Um seine Punkte zu deaktivieren, drücken Sie ESC oder starten Sie den Befehl PunkteAus.


Kontrollpunkte auf einer Zone

Einfügeoptionen und -parameter

Die Zonenobjekt-Einfügungsoptionen und -Parameter sind in unterschiedliche Reiter aufgeteilt und über verschiedene Dialoge abrufbar:

  • Verzonen-Einfügungsdialog (nur beim Einfügen einer neuen Zone verfügbar).
  • Abschnitt der Zone-Eigenschaften, im Lands-Design-Bearbeitungspanel.
  • Zone-Objekteigenschaftendialog.

Allgemein

Allgemeine Attribute für das Zone-Objekt: Name, Farbe, Ebene, Code und Stichwörter.

Zone

  • Etikett: zeigt den Text und Bereich der Zone an. Verwenden Sie das Kontrollkästchen, um es ein- oder auszublenden.
    • Etikettenbox: mehrzeilige Textkästchen zur Eingabe des Etikettentexts.
    • Flächeninhalt anzeigen: Option zur Angabe der Quadratmeter im Etikett.
    • Textstil.
    • Höhe: Textgröße.
  • Schraffierungsattribute:
    • Pattern
    • Farbe
    • Winkel
    • Skala

Material

Mit diesem Reiter können Sie das diesem Element zugewiesene texturierte Bild ändern und Bildgröße und Reflexionsvermögen definieren.

Bearbeitungsoptionen

Dies sind die Bearbeitungsoptionen für das Zone-Objekt, verfügbar im Bearbeitungsbereich des Bearbeitungsfensters:

  • Aktualisieren
  • In 2D umwandeln. Zeigt das Objekt in der 2D-Darstellung an.
  • In 3D umwandeln. Zeigt das Objekt in der 3D-Darstellung an.
  • 2D und 3D anzeigen: Aktiviert gleichzeitig die 2D- und 3D-Darstellung des Objekts.
  • Eigenschaften von einem anderen Objekt kopieren
  • Anpassen
    • Erhöhung
    • Auf Nullhöhe
    • An Gelände
  • Nicht an Gelände anpassen

Grafische Zone-Darstellung:

Das Zone-Objekt wird als Schraffierungsmuster in der 2D-Darstellung und als texturierte Fläche in 3D angezeigt. Das Textetikett der Zone wird in beiden Anzeigemodi gleich dargestellt.
