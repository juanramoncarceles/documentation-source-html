Parameter

Parameter werden verwendet, um Geometrie-Objekten Informationen hinzuzufügen. Diese Objekte können entweder Lands-Design-Elemente sein (Pflanzen. Zäune, Pfade, Gelände etc.) oder Rhino-Elemente (Punkte, Kurven, Flächen, Flächenverbände, Extrusionen, Polygonnetze und Blöcke).

Parameter können auf zwei verschiedene Weisen erzeugt und bearbeitet werden:

  • Nach Dokument
  • Nach Objekt einzeln

Die vorhandenen Parameter sind für jede Objekteinheit im Rhino-Eigenschaftenpanel sichtbar, unter dem Parameter-Symbol , von wo aus die Werte hinzugefügt werden können.


Parameter im Eigenschaftenpanel

Parameter nach Dokument

Wo befindet sich dieser Befehl?
Tastatur:
Dokumenteigenschaften
Menü:
Werkzeuge > Optionen > Dokumenteigenschaften > Parameter

Die Dokumentparameter werden im Dialog der Rhino-Dokumenteigenschaften > Abschnitt Parameter verwaltet. Diese Parameter sind für jedes einzelne Geometrie-Objekt im Modell verfügbar.

Die vorhandenen Parameter werden in einer Liste mit ihrer jeweiligen Kategorie und Werttyp angezeigt.


Abschnitt Parameter in den Rhino-Dokumenteigenschaften

  • Hinzufügen: öffnet den Parameter-Eigenschaftendialog um einen neuen Parameter zu erzeugen.
  • Bearbeiten: öffnet den Parameter-Eigenschaftendialog um einen aus der Liste ausgewählten Parameter zu bearbeiten.
  • Entfernen: entfernt einen ausgewählten Parameter von der Liste.

Die Information benutzerdefinierter Parameter, die Lands-Design-Elementen nach Dokument hinzugefügt wurde, kann mit Lands Designs Auflistungswerkzeugen aufgelistet werden.

Parameter nach Objekt

Objektparameter sind diejenigen, die für ein einzelnes Objekt gesondert verwaltet werden. Ihre Erzeugung, Bearbeitung und Entfernung erfolgt im Rhino-Eigenschaftenpanel , unter dem Parametersymbol nach Auswahl eines beliebigen Objekttyps.

  • Öffnet den Parameter-Eigenschaftendialog, damit Sie einen neuen Parameter erzeugen können.
  • Öffnet den Parameter-Eigenschaftendialog, damit Sie den Parameter bearbeiten können. Sie müssen den Namen des Parameters auswählen, um diese Schaltfläche zu aktivieren.
  • Löscht einen vorhandenen Parameter. Sie müssen den Namen des Parameters auswählen, um diese Schaltfläche zu aktivieren.

Hinweis: Name und Beschreibung sind native Parameter und können nicht bearbeitet werden.

Parameter-Eigenschaftendialog


Parameter-Eigenschaftendialog

  • Name: Name des Parameters.
  • Kategorie: der Abschnitt, in dem die Parameter zu sehen sein werden. Sie können entweder eine neue Kategorie eingeben oder eine der vorhandenen Kategorien auswählen (Allgemein, Anzeige, Geometrie und Standort), zuzüglich zu den eventuell erstellten neuen Kategorien.
  • Datentyp: zeigt das Wertformat an. Wählen Sie eine der Optionen in der Liste aus.
  • Beschreibung: Feld zum Hinzufügen von Anmerkungen über den Parameter.
