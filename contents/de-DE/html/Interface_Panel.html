Das Bearbeitungspanel von Lands Design ist ein andockbares Panel, das für alle Objekte im Modell Optionen zur Bearbeitung in Echtzeit bietet.

Im Bearbeitungspanel können Sie:

  • Die Eigenschaften der ausgewählten Objekte bearbeiten
  • Die gleichen Funktionen wie im Kontextmenü für die aktuelle Auswahl ausführen
  • Den Anzeigemodus von 3D-Pflanzen ändern
  • Ebenenstati anwenden
  • Der Zeichnung Filter anwenden

Das Bearbeitungspanel ist in vier Abschnitte oder Bereiche aufgeteilt:

  1. Werkzeugleiste
  2. Liste zur Objektauswahl
  3. Bearbeitungsbereich
  4. Bereich der Objekteigenschaften


Lands-Design-Bearbeitungspanel (wenn keine Objekte ausgewählt sind).

1. Werkzeugleiste

Die Zeile der Werkzeugleiste im Bearbeitungspanel verfügt über folgende Optionen.

  • Schaltflächen für 2D- und 3D-Darstellung: zum Hin- und Herschalten zwischen der Darstellung in 2D und 3D aller Lands-Design-Objekte im Modell. Dieser Vorgang wirkt sich auf alle Ansichtsfenster aus.
  • Schaltfläche Auswählen: zeigt eine Liste aller verschiedenen Elementtypen in Lands Design an, in der Sie eine Auswahl von Elementen des gleichen Typs vornehmen können.
  • An Gelände anpassen: ermöglicht die automatische Platzierung von Objekten auf einer Geländefläche während des Einfügens.
  • Aktualisieren: aktualisiert automatisch jede auf ein Lands-Design-Objekt angewandte Änderung.

2. Auswahlliste

In diesem Bereich sind die im Modell ausgewählten Objekte zu sehen.

Wenn Keine Objekte ausgewählt sind, zeigt die Liste die verschiedenen im Modell vorhandenen Vegetationsarten und Stadtmöbelblöcke an, wie auch die Zahl jeweils verfügbarer Elemente.

Durch Auswahl einer Pflanzenart aus dieser Liste wählen Sie alle zur gleichen Art gehörenden Pflanzen im Modell auf einmal aus.

Durch Doppelklick auf das Symbol neben der Art können Sie alle Pflanzen der gleichen Art anzeigen oder verbergen.

  • Die Pflanzenarten sind sichtbar.
  • Die Pflanzenarten sind verborgen.


Bereich Auswahlliste im Bearbeitungspanel.

3. Bearbeitungsbereich

In diesem Bereich sind die für die aktuelle Auswahl verfügbaren Bearbeitungsbefehle zu sehen. (Diese Befehle sind auch im Kontextmenü für die aktuelle Auswahl zu finden.)


Bearbeitungsbereich im Bearbeitungspanel (für einen Wald).

4. Eigenschaftenbereich

Dieser Bereich ist in Reiter aufgeteilt, welche die unterschiedlichen Eigenschaften und Optionen für jeden Objekttyp anzeigen. (Diese Eigenschaften sind auch im Objekteigenschaftendialog verfügbar.)

Wenn ein Objekt ausgewählt ist, zeigt dieser Bereich die Dokumenteigenschaften an.

Wenn die Schaltfläche Alle Änderungen anwenden aktiviert ist, wird automatisch jede auf das bearbeitete Objekt angewandte Änderung aktualisiert.


Eigenschaftenbereich im Bearbeitungspanel (für eine Auswahl von Pflanzen).
