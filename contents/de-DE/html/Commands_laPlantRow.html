Pflanzenreihe einfügen

Durch Ausführen des Befehls laPflanzenreihe wird der Einfügedialog für Pflanzenreihe geöffnet. Dieser Dialog zeigt die grundlegenden Optionen des Pflanzenreihenobjekts auf verschiedenen Reitern an:

  • Reihe: Parameter der Pflanzenreihe.
  • Pflanze: Parameter der Pflanzenart-Elemente in der Pflanzenreihe.
  • Format: Format der Pflanzenarten in der Reihe.
  • Rendern: Parameter zur Definierung, wie Pflanzenarten in Pflanzenreihen in Renderdarstellungen aussehen.

Vorgehen:

  1. Wählen Sie eine Pflanzenart aus der Liste, unter dem Reiter Pflanze. Klicken Sie auf die Schaltfläche Durchsuchen... um eine neue aus der Pflanzendatenbank auszuwählen. Wählen Sie die restlichen Pflanzenparameter aus.
  2. Wählen Sie die Reihenparameter unter dem Reiter Reihe. Klicken Sie auf OK, um das Dialogfeld zu schließen.
  3. Wählen Sie eine im Modell vorhandene Kurve aus, um den Pflanzenreihenpfad zu definieren, oder wählen Sie eine unterschiedliche Option in der Befehlszeile.
    • Polylinie: die Reihe wird so erzeugt, als ob Sie eine Polylinie zeichnen würden.
    • Spline: die Reihe wird so erzeugt, als ob Sie einen Spline zeichnen würden.
    • Kreis: die Reihe wird so erzeugt, als ob Sie einen Kreis zeichnen würden. Mit dem ersten Klick wird die Kreismitte bestimmt, und mit dem zweiten Klick der Radius.
    • Bogen: die Reihe wird so erzeugt, als ob Sie einen Bogen zeichnen würden. Mit dem ersten Klick wird der Einfügungspunkt bestimmt. Der zweite Klick bestimmt einen Mittelpunkt im Bogen. Mit dem dritten und letzten Klick wird bestimmt, wo der Bogen endet.
    • Rechteck: die Reihe wird so erzeugt, als ob Sie ein Rechteck zeichnen würden. Mit dem ersten Klick wird eine Ecke des Rechtecks bestimmt, und mit dem zweiten Klick die gegenüberliegende Ecke.
  4. Beenden Sie den Befehl durch Drücken von EINGABE, ESC oder Klicken auf die rechte Maustaste.

Die Reihenparameter können während ihrer Einfügung und nach Beendigung des Befehls bearbeitet werden, und zwar im Lands-Design-Bearbeitungspanel, im Abschnitt Eigenschaften, unter dem Reiter Reihe.

Kontrollpunkte

Pflanzenreihen haben dieselben Kontrollpunkte wie die zu ihrer Erzeugung verwendeten Kurven, ungeachtet der Zahl der Pflanzenelemente oder ihrer Position.

Zuzüglich haben sie einen Kontrollpfeil zum Verschieben des gesamten Objekts.

Pfadreihe-Kontrollpunkte werden bei der Einzelauswahl von Pflanzenreihen automatisch aktiviert. Um einen oder mehrere Pflanzenreihen-Objektkontrollpunkte zu aktivieren, wählen Sie einfach die Pflanzenreihen aus und starten Sie den Befehl PunkteEin (oder drücken Sie F10). Um die Punkte zu deaktivieren, drücken Sie ESC oder starten Sie den Befehl PunkteAus.

Einfügeoptionen und -parameter

Die Pflanzenreihen-Einfügungsoptionen und -Parameter sind in unterschiedliche Reiter aufgeteilt und über verschiedene Dialoge abrufbar:

  • Pflanzenreihe-Einfügungsdialog (nur beim Einfügen neuer Pflanzenreihen verfügbar).
  • Abschnitt der Pflanzenreihen-Eigenschaften, im Lands-Design-Bearbeitungspanel.
  • Pflanze-Objekteigenschaftendialog.

Allgemein

Allgemeine Attribute für das Pflanzenobjekt: Name, Farbe, Code und Stichwörter.

Reihe

Zum Platzieren der Pflanzen in Reihen sind zwei Optionen verfügbar:

  • Anzahl Einheiten: zur Bestimmung der Gesamtmenge von Pflanzenelementen entlang der Reihen-Pfadkurve.
  • Trennung: zur Platzierung der Pflanzenelemente in einem gewissen Abstand, gemäß einer Ausrichtung auf der Reihenpfadkurve (am Anfang, zentriert oder am Ende).

Pflanzenreihen können verschiedene Pflanzenarten kombinieren:

  • Anzahl Elemente: Geben Sie die Anzahl verschiedener Pflanzenarten ein. Die Liste verschiedener Arten wird in der Liste zur Objektauswahl im Lands-Bearbeitungspanel erscheinen, wo sie zur Bearbeitung ihrer Parameter ausgewählt werden können.
  • Verteilung: Optionen zum Verteilen der Pflanzenarten wenn mehr als ein Typ vorhanden ist.

Eingefügte Einheiten: zeigt die genaue Anzahl von Elementeinheiten in der Pflanzenreihe an.

Pflanze

  • Art: Der Name der ausgewählten Pflanzenart. Die Zusammenstellung zeigt eine Liste der verschiedenen Pflanzenarten an, die bereits im aktuellen Modell eingesetzt wurden. Sie können eine Pflanzenart aus der Liste auswählen, oder aber eine neue aus der Pflanzendatenbank, wofür Sie auf die Schaltfläche Durchsuchen... klicken.
  • Bemaßungen in der Zeichnung:
    • Kronendurchmesser
    • Ungefähre Höhe
    • Alter
  • Natürliche Variation: wenn sie aktiviert ist, setzt diese Option eine willkürliche Höhenvariation für die Pflanzenart in der Reihe. Der Maximalwert (100 %) erzeugt Pflanzenarten, die von 50 % bis zu 150 % ihrer Größe reichen.
  • Willkürliche Drehung: wenn sie aktiviert ist (Standard), stellt diese Option eine willkürliche Drehung der Pflanzen ein. Ist sie deaktiviert, können Sie für jede Pflanze einen spezifischen Drehwinkel definieren. Der Drehwinkel wird von der vertikalen Pflanzenachse aus gemessen.

Die minimalen und maximalen Werte der Pflanzenhöhe (Alter der Pflanze) werden durch die Eigenschaften jeder Pflanzenart definiert und können im Eigenschaftendialog der Pflanzenarten bearbeitet werden, der in der Pflanzendatenbank verfügbar ist.

Wenn Pflanzenreihen mehr als eine Art enthalten, sind die Parameter jeder einzelnen davon verfügbar, und zwar nach Auswahl der gewünschten Art von der Objektauswahl-Liste im Lands-Bearbeitungspanel.

Format

  • Kaliber: Rindenbemaßung der Pflanzenart. Die verfügbaren Werte reichen von 10 bis 130 mm, Sie können aber auch jeden sonstigen Wert eingeben.
  • Darstellungsmodus: verfügbar sind drei Darstellungsmodi, Sie können aber auch neue eingaben.
    • Nackte Wurzel
    • Wurzelballen
    • Behälter
    •

    Rendern

    Dieser Reiter bestimmt, wie die Pflanzenarten in Pflanzenreihen in einer Renderdarstellung aussehen. Sichtbar ist dieser Reiter nur, wenn die 3D-Darstellung der Pflanzen in den Dokumenteigenschaften auf Realistisch eingestellt ist.

    • Jahreszeit: Wenn das Kästchen Globale Jahreszeit ignorieren markiert ist, können Sie auswählen, in welcher Jahreszeit die Pflanzen gerendert werden: Vorfrühling, Frühling, Spätfrühling, Sommer, Spätsommer, Herbst, Winter...
    • Laubdichte: Je nach Definition erzeugt Flamingo nXt mehr oder weniger Blätter.

    Bearbeitungsoptionen

    Dies sind die Bearbeitungsoptionen für das Pflanzenreihenobjekt, verfügbar im Bearbeitungsbereich des Bearbeitungsfensters:

    • Aktualisieren
    • Gruppierung aufheben
    • Eigenschaften von einem anderen Objekt kopieren
    • Auf eine andere Kurve kopieren
    • ZufälligFestlegen
    • Kurve umkehren
    • Kurven extrahieren
    • Anpassen
      • Erhöhung
      • Auf Nullhöhe
      • An Gelände
    • Nicht an Gelände anpassen
    • Pflanze
      • Art bearbeiten
      • Etikett zuweisen (Dieser Befehl erscheint nur, wenn den Pflanzen in der Reihe kein Etikett zugewiesen wurde)
      • Verknüpfend markieren.
      • Etikett erkennen (dieser Befehl erscheint nur, wenn es ein die Pflanzenart im Pflanzenreihenobjekt referenzierendes Pflanzenetikett gibt)
      • Foto einfügen

    Grafische Anzeige der Pflanzenreihe

    Pflanzenarten in Reihen werden grafisch genauso dargestellt wie Pflanzenarten die einzeln eingefügt wurden. Die grafische Darstellung von Pflanzenarten in Reihen kann in den Dokumenteigenschaften geändert werden.
