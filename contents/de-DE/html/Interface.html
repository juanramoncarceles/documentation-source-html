Übersicht über die Benutzeroberfläche

Die meisten Lands-Design-Befehle sind in verschiedene Werkzeugleisten gruppiert.

Sie können Befehle über die Schaltflächen der Werkzeugleiste, das Dropdown-Menü von Lands Design oder durch Eingabe von Befehlsverweisen in der Befehlszeile ausführen.

Die meisten Befehle verfügen über Einfügedialoge und Dialogfenster, die Sie durch die Ausführung begleiten.

Die folgenden Lands-Design-Elemente werden auf dem Bildschirm angezeigt, wenn Sie eine neue Rhino-Zeichnung unter Verwendung einer Lands-Design-Vorlage öffnen.


Die Benutzeroberfläche von Lands Design in Rhino

  1. Lands Design Menü
  2. Befehlszeile: Eine grün hervorgehobene Nachricht in der Befehlszeile zeigt an, dass Lands Design geladen wurde. Die Nachricht weist darauf hin, dass Tibidabo geladen und Lands Design installiert wurde. Zudem wird das Datum des Builds angegeben.
  3. Lands-Design-Hauptwerkzeugleiste
  4. Objekteigenschaften und Ebenenmanager-Dialog
  5. Lands Design-Bearbeitungspanel und Animationsfenster
