Einfügen eines Pfads:

Durch Ausführen des Befehls laPfad wird der Einfügedialog für Pfad geöffnet. Dieser Dialog zeigt die grundlegenden Optionen des Pfadobjekts auf zwei Reitern an.

  • Pfad: Pfadparameter.
  • Material: Materialeinstellungen für das Pfadobjekt.

Vorgehen:

  1. Wählen Sie das Pfad-Schnittprofil aus. Eins standardmäßiges rechteckiges Profil (von 3 x 0.2 Meter) ist bereits zugewiesen. Stattdessen können Sie auch eine im Modell vorhandene Kurve auswählen (Auswahl-Option) oder sie im Modell zeichnen (Option Nach Punkt auswählen).
  2. Wählen Sie die restlichen Pfadparameter aus . Klicken Sie auf OK, um das Dialogfeld zu schließen
  3. Wählen Sie eine im Modell vorhandene Kurve aus, um den Pfad zu erzeugen, oder wählen Sie eine unterschiedliche Option in der Befehlszeile.
    • Polylinie: der Pfad wird so erzeugt, als ob Sie eine Polylinie zeichnen würden.
    • Spline: der Pfad wird so erzeugt, als ob Sie einen Spline zeichnen würden.
    • Kreis: der Pfad wird so erzeugt, als ob Sie einen Kreis zeichnen würden. Mit dem ersten Klick wird die Kreismitte bestimmt, und mit dem zweiten Klick der Radius.
    • Bogen: der Pfad wird so erzeugt, als ob Sie einen Bogen zeichnen würden. Mit dem ersten Klick wird der Einfügungspunkt bestimmt. Der zweite Klick bestimmt einen Mittelpunkt im Bogen. Mit dem dritten und letzten Klick wird bestimmt, wo der Bogen endet.
    • Rechteck: der Pfad wird so erzeugt, als ob Sie ein Rechteck zeichnen würden. Mit dem ersten Klick wird eine Ecke des Rechtecks bestimmt, und mit dem zweiten Klick die gegenüberliegende Ecke.
  4. Beenden Sie den Befehl durch Drücken von Eingabe, Esc oder Klicken auf die rechte Maustaste.

Hinweis: vergewissern Sie sich, dass die als Pfad-Schnittprofil ausgewählte Kurve an der XY-Achse ausgerichtet ist.

Kontrollpunkte

Pfadobjekte verfügen über drei verschiedene Typen von Kontrollpunkten:

  1. Pfadachse aus Kontrollpunkten: dies sind die selben Kontrollpunkte wie die aus den Kurven, die zum Erzeugen der Pfade verwendet werden.
  2. Kontrollpfeil zum Verschieben: Am Anfang des Pfadobjekts befindlicher Pfeil, der zum Verschieben des gesamten Objekts verwendet wird.
  3. Schnittprofil-Kontrollpunkte: Diese Kontrollpunkte werden verwendet, um das Pfadschnittprofil zu verändern. Dies sind die selben Kontrollpunkte wie die aus den Kurven, die zum Definieren des Pfadschnittprofils verwendet werden. Sie befinden sich Am Anfang des Pfadobjekts.

Pfad-Kontrollpunkte werden bei der Einzelauswahl von Pfaden automatisch aktiviert. Um einen oder mehrere Pfad-Kontrollpunkte zu aktivieren, wählen Sie einfach die Pfade aus und starten Sie den Befehl PunkteEin (oder drücken Sie F10). Um die Punkte zu deaktivieren, drücken Sie ESC oder starten Sie den Befehl PunkteAus.

Einfügungsoptionen und -parameter

Die Pfadobjekt-Einfügungsoptionen und -Parameter sind in unterschiedliche Reiter aufgeteilt und über verschiedene Dialoge abrufbar:

  • Pfad-Einfügungsdialog (nur beim Einfügen neuer Pfade verfügbar).
  • Abschnitt der Pfadeigenschaften, im Lands-Design-Bearbeitungspanel.
  • Pfad-Objekteigenschaftendialog.

Allgemein

Allgemeine Attribute für das Pfadobjekt: Name, Farbe, Ebene, Code und Stichwörter.

Pfad

  • Schnittprofil: zum Definieren des Pfadschnittprofils verwendete Kurve.
  • Glätte: Präzision, mit der das Pfadobjekt entlang der Achsenkurve unterteilt ist, um mit seinem Pfad übereinzustimmen. Dieser Parameter ist nur dann relevant, wenn das Pfadobjekt aus einer gebogenen Kurve erzeugt wurde.
  • Ausrichtung: markiert die Position des Pfadschnittprofils, von wo aus es entlang der Pfadachsenkurve extrudiert wird. Es sind neun Ausrichtungen möglich. Keine Ausrichtung behält die Ausrichtung des Pfads entlang der Pfadachse bei, selbst wenn Sie das Pfadschnittprofil ändern.

Material

Mit diesem Reiter können Sie das diesem Element zugewiesene texturierte Bild ändern und Bildgröße und Reflexionsvermögen definieren.

Bearbeitungsoptionen

Dies sind die Bearbeitungsoptionen für das Pfad-Objekt, verfügbar im Bearbeitungsbereich des Bearbeitungsfensters:

  • Aktualisieren
  • In 2D umwandeln. Zeigt das Objekt in der 2D-Darstellung an.
  • In 3D umwandeln. Zeigt das Objekt in der 3D-Darstellung an.
  • 2D und 3D anzeigen: Aktiviert gleichzeitig die 2D- und 3D-Darstellung des Objekts.
  • Eigenschaften von einem anderen Objekt kopieren
  • Auf eine andere Kurve kopieren
  • Kurve umkehren
  • Kurve extrahieren
  • Anpassen
    • Erhöhung
    • Auf Nullhöhe
    • An Gelände
  • Nicht an Gelände anpassen
  • Pfad
    • Schnittprofil bearbeiten: der Befehl platziert eine mit dem Pfadschnittprofil ausgerichtete Konstruktionsebene am Anfang des Objekts. Durch seine Kontrollpunkte erleichtert es die Bearbeitung des Pfadprofils. Führen Sie den Befehl KEbene aus (und wählen Sie Welt > Drauf in der Befehlszeile) aus, um die Position der Konstruktionsebene wiederherzustellen.
