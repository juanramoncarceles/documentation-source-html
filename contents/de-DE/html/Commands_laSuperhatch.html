Superschraffierung

Wo befindet sich dieser Befehl?
Tastatur:
laSuperschraffierung
Menü:
Lands Design > Dokumentation > Superschraffierung

Mit dem Befehl Superschraffierung wird ein texturiertes Flächenelement, basierend auf einer Hintergrundbild-Wiederholung, erstellt.

  • Superschraffierung einfügen
  • Superschraffierung-Kontrollpunkte
  • Einfügeoptionen und -parameter
  • Graphische Darstellung

Vorgehen:

  1. Führen Sie den Befehl laSuperschraffierung aus. Wählen Sie eine der beiden Optionen in der Befehlszeile aus, um das Superschraffierungsobjekt einzufügen:
    • Inneren Punkt: innerhalb einer geschlossenen Begrenzung von 2D-Kurven klicken.
    • Begrenzungskurve: geschlossene und planare Kurve auswählen.
  2. Nach dem ersten Schritt erscheint der Einfügungsdialog für Superschraffierung. Dieser Dialog zeigt die grundlegenden Optionen des Superschraffierung-Objekts auf einem einzigen Reiter an:
    • Superschraffierung: Superschraffierungsparameter.
    Wählen Sie die grundlegenden Einfügungsoptionen. Schließen Sie den Dialog durch Drücken von Eingabe, Esc oder Klicken auf die rechte Maustaste.


Superschraffierung.



Kontrollpunkte

Das Superschraffierungsobjekt hat die gleichen Kontrollpunkte wie die Kurven, aus denen es erzeugt wurde, oder die während des Einfügens aus den Begrenzungen des gewählten Punkts erzeugte Begrenzungskurve.

Zuzüglich hat es einen Kontrollpfeil zum Verschieben der gesamten Fläche.

Um einen Superschraffierung-Kontrollpunkt zu aktivieren, führen Sie den Befehl PunkteEin aus (oder drücken Sie F10) und wählen Sie das Superschraffierung-Objekt. Um seine Punkte zu deaktivieren, drücken Sie ESC oder starten Sie den Befehl PunkteAus.

Einfügeoptionen und -parameter

Die Superschraffierung-Einfügungsoptionen und -Parameter sind in unterschiedliche Reiter aufgeteilt und über verschiedene Dialoge abrufbar:

  • Superschraffierung-Einfügungsdialog (nur beim Einfügen einer neuen Superschraffierung verfügbar).
  • Abschnitt der Superschraffierung-Eigenschaften, im Lands-Design-Bearbeitungspanel.
  • Superschraffierung-Objekteigenschaftendialog.

Allgemein

Allgemeine Attribute für das Superschraffierung-Objekt: Name, Farbe, Ebene, Code und Stichwörter.
