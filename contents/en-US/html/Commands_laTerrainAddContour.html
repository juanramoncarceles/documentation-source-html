Terrain Add Contour

Where can I find this command?
Keyboard:
laTerrainAddContour
Icon:
left click
Toolbar: Lands Terrain
Menu:
Lands Design > Terrain Modeling > Terrain > Add Contour
Edit Panel:
Select a terrain > Edit area in Edit Panel > Terrain > Add Contour

This command adds a new input data to an existing terrain. This input data can be either 3D curves, contour lines or point clouds.

Steps:

  1. Pick a terrain.
  2. Select the new terrain input data (3D curves, contour lines or point clouds) to add to the terrain.


Existing terrain with the curve to add as the new contour line.


Terrain after having added a new contour line.

The new contour line added (as any other contour line or input data) can be removed from the elevation data list, in the Terrain Edit area of the Edit panel and Properties dialog, under the Input data tab, after selecting the terrain.
