Copy to another curve

Where can I find this command?
Keyboard:
laCopyToCurve
Menu:
Lands Design > Edit > Copy to another curve
Edit Panel:
Select a Lands element > Edit area in Edit Panel > Copy to another curve

The laCopyToCurve command creates new Lands elements by selecting existing ones and applying their properties to new curves.

This command is available from the Edit area in Edit panel, for the following objects:

  • Plant row
  • Forest
  • Dwarf Wall - Hedge
  • Fence
  • Row
  • Path
  • Stair
  • Zone

Steps:

  1. Select the Lands element whose parameters you want to apply on a new object.
  2. Pick an existing curve in the model to define the new object path or boundary. Or select a different option in the command line to draw the new object according to different curves:
    • Polyline: the object will be created as if you were drawing a polyline.
    • Spline: the object will be created as if you were drawing a spline.
    • Circle: the object will be created as if you were drawing a circle. The first click specifies the center of the circle, and the second click specifies the radius.
    • Arc: the object will be created as if you were drawing an arc. The first click specifies the insert point. Second click specifies a middle point in the arc. The third and the last click specify where the arc ends.
    • Rectangle: the object will be created as if you were drawing a rectangle. The first click specifies one corner of the rectangle and the second click specifies the opposite corner.
