Tag object as Terrain

Where can I find this command?
Keyboard:
laTagObjectAsTerrain
Menu:
Lands Design > Terrain modeling > Tag Object As Terrain
Edit Panel:
Select a Rhino object > Edit area in Edit panel > Tag object as terrain

The laTagObjectAsTerrain command turns a piece of geometry into a Terrain. It won't be a true Lands Design Terrain, but plants and other elements will detect it as a terrain and will be projected on top of its upper surface.

This command is available for any Rhino geometry and the Superhatch object.

You can undo this operation by running the laUntagObjectAsTerrain command, available from the Edit area in the Edit panel after selecting the object.
