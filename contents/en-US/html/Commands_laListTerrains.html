Terrain List

Where can I find this command?
Keyboard:
laListTerrains
Icon:
left click [long click]
Toolbar:
Menu:
Lands Design > Documentation > Listing > Terrain list

The laListTerrains command inserts a 2D table displaying the information of the existing terrains in the model, and the operations created on them.

The features of terrain lists are the same as the rest of listing object tools.

  • Insert a terrain list
  • Insert options and parameters
  • Edit options
  • Graphical display


Terrain list.

Terrain list fields

These are the default fields for the terrain list object, available under the Element (Terrain) list tab:

  • Name: name given to the terrain object.
  • Index: terrain identifier (ID).
  • Area: flat area of the operation applied on the terrain.
  • Volume from fills: amount of land movement added to the terrain.
  • Volume from cuts: amount of land movement subtracted from the terrain.
