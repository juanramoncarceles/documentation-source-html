Edit

Where can I find this command?
Keyboard:
laEdit
Icon:
left click]
Menu:
Lands Design > Edit > Edit...
Shortcut:
Double-click on the object to edit

The laEdit command opens the Properties Explorer dialog for a selection of Lands Design objects and allows you to edit them. This dialog is similar to the Properties dialog box, but it displays the list of the objects selected.

If no objects are selected, the Properties dialog shows the Document Properties.

When more than one Lands Design object is selected, the Properties dialog displays the list of the selected entities, grouped by element type (plant, terrain, forest, shrub, etc...)
