The laListPipes command inserts a 2D table displaying the information of the existing pipes in the model.

The features of urban furniture blocks lists are the same as the rest of listing object tools.

  • Insert a pipe list
  • Insert options and parameters
  • Edit options
  • Graphical display


Pipe list.

Pipe list

These are the default fields for the pipe list object, available under the Element (Pipe) list tab:

  • Pipe icon: small pipe drawing displayed in the list.
  • Index: pipe identifier (ID). Lands assigns an ID value to pipes by default.
  • Section (diameter): pipe diameter dimension.
  • Pressure: water pressure inside pipes measured in atm.
  • Material: material description.
  • Description: text displayed in the pipe description field.
  • Linear meters: pipe length measured in linear meters.
