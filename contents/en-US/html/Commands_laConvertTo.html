Convert To

Where can I find this command?
Keyboard:
laConvertTo
Icon:
left click
Menu:
Lands Design > Edit > Convert To > 2D / 3D / 2D and 3D
Edit Panel:
Select a Lands element > Edit area in Edit Panel > Convert to 2D / Convert to 3D / Show 2D and 3D

The laConvertTo command changes the representation of Lands elements according to different options:

Steps:

  1. Select the object to change its representation.
  2. Select one of the following options on the command line:
    • 2D: replaces the 3D representation of a Lands Design object by its 2D representation. (This option is only effective when the block is displayed in 3D)
    • 3D: replaces the 2D representation of a Lands Design object by its 3D representation. (This option is only effective when the block is displayed in 2D)
    • All: shows the 2D and 3D representation of a Lands Design object simultaneously.
