Zones List

Keyboard:
laListZones
Icon:
left click
Toolbar:
Menu:
Lands Design > Documentation > Listing > Zoning list

The laListZones command inserts a 2D table displaying the information of the existing zones in the model.

The features of zoning lists are the same as the rest of listing object tools.

  • Insert a zoning list
  • Insert options and parameters
  • Edit options
  • Graphical display


Zoning list.

Zoning list fields

These are the default fields for the zone list object, available under the Element (Zoning) list tab:

  • Sample icon: small preview of the hatch pattern used for the zone 2D display. You can edit the scale value to control the icon size.
  • Index: zone identifier (ID). Lands assigns an ID value to zones by default.
  • Name
  • Area
