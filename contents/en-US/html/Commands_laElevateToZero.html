Elevate To Zero

Where can I find this command?
Keyboard:
laElevateToZero
Icon:
left click
Menu:
Lands Design > Edit > Adjust > To Zero Elevation
Edit Panel:
Select a Lands element > Edit area in Edit Panel > Adjust > To zero elevation

The laElevateToZero command moves any kind of object vertically to the zero elevation of the document.

If the object to move is not flat or is composed by different items, such as a Forest or a Row, the lower point of this object will be moved to the zero elevation.

Steps:

  1. Select the object to elevate to zero.
