Set Viewpoint

Where can I find this command?
Keyboard:
laSetViewpoint
Icon:
left click
Toolbar: Lands Tools
Menu:
Lands Design > Set viewpoint

The laSetViewpoint command sets the camera viewpoint from the viewer and target position.

Steps

  1. Pick the viewer point in the viewport where you want to set the viewer.
  2. Pick the target point where the viewer will look at.
