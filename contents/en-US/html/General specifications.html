General Specifications

Software

Lands Design requires Rhinoceros 6 (SR14 or later) or Rhino 7 to run.

Versions and Updates

Lands Design checks for updates automatically. If there is an update available, an icon appears in the Windows taskbar notification area.

Languages

Lands Design's interface, templates and toolbars are available in 7 languages: English, Spanish, French, Italian, German, Czech, and Chinese (simplified).

A dialog to choose Lands Design's language can be found in the Lands Design Options dialog or the Lands Design section inside the Rhino Options dialog.

Units

Lands Design supports Metric and Imperial unit systems. When changing the drawing units, Lands Design shows a message asking if you want to scale the model according to the unit conversion factor.

Layers

Lands Design allows you to set the default layer in the creation commands. The Layers panel can be found in the Lands Design Options dialog or the Lands Design section inside the Rhino Options dialog.

When running the object creation commands, the current layer is switched to the target layer. When the commands end, the previously active layer becomes the current layer, unless the user changes it during the command.

Help

Lands Design includes some Help options, which can be found in the Lands Design main menu.
Options include the Help file, links to Technical Support and Lands Design website, and the About dialog (laAbout command) with information about License, User Name, Organization and Serial Number.
