Object Properties Dialog

Keyboard:
laProperties
Shortcut:
Double-click on the object to edit

The Object Properties dialog is a floating dialog that allows you to edit the selected Lands Design object properties.

To open the the Properties dialog, run the laProperties command and select the object or objects to edit. Hit Enter or righ-click to end the selection.


Lands Design Object Properties dialog.

The Object Properties dialog is organized in different tabs, depending on the object type.

Tabs

  • General:
    All Lands Design elements have this tab. It determines the general attributes of the selected object, such as the Name, Color, Layer, Code and Keywords.
  • Render:
    Plants, Plant rows, Forests and Shrubs have this tab. It determines how the plant species will be displayed in a render file, according to the season and their foliage density.
  • Plant:
    Plants, Plant rows, Forests and Shrubs have this tab. It determines the Plant species properties.
  • Row:
    Plant rows and Rows have this tab. It determines the Plant rows and Rows parameters.
  • Groundcover:
    Only Groundcover objects have this tab. It determines the Groundcover parameters.
  • Dwarf wall/Hedge:
    Only Dwarf wall/Hedge objects have this tab. It determines the Dwarf wall/Hedge parameters.
  • Fence:
    Only Fence objects have this tab. It determines the Fence parameters.
  • Stair:
    Only Stair objects have this tab. It determines the Stair parameters.
  • Path:
    Only Path objects have this tab. It determines the Path parameters.
  • Terrain:
    Only Terrains have this tab. It determines the Terrain parameters.
