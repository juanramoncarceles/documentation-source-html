Edit species

Where can I find this command?
Keyboard:
laPlantEditSpecies
Icon:
left click]
Menu:
Lands Design > Edit > Edit Species
Edit Panel:
Select a Lands vegetation element > Edit area in Edit Panel > Plant > Edit species

The laPlantEditSpecies command selects all plants of the same species in the model and lets you change their common features (name, height and representation modes) from the Object properties area in the Lands Edit panel.

This command is available from the Edit area in Edit panel, for the following objects:

  • Plant
  • Plant row
  • Forest
  • Shrub
  • Groundcover
