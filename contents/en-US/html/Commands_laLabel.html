The Label command draws an annotation leader and a label with a text.

Insert a label:

Steps:

  1. Pick the head of the leader.
  2. Pick the position of the label.


Label object.



Control points

Label object has two control points: one at the arrowhead and another at the center of the Label.

To turn Label control points on, run the PointsOn command (or press F10) and select the Label. To turn the points off, press ESC button or run the PointsOff command.

Parameters

The Label insert options and parameters are divided in different tabs, and are available from different dialogs:

  • Label properties section, in the Lands Design Edit panel.
  • Object Properties dialog.
  • Properties Explorer dialog.

General

General attributes for the label: Name, Color, Layer, and Transparency (only visible in Conceptual display mode).

Label

This tab shows the parameters related to the label.

  • Shape: display options of the text in the label.
  • Font: text attributes and parameters.

Leader

  • Line: parameters of the leader line.
  • End: parameters of the leader pointer.

Edit options

These are the edit options for the label object, available in the Edit area of the Edit panel:

  • Copy properties from another object.
  • Adjust
    • Elevation
    • To zero elevation
    • To terrain
  • Don't adjust to terrain

Graphical display

The Label object only has a 2D representation and it is visible with any display mode.
