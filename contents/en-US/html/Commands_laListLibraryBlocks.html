Library blocks List

Where can I find this command?
Keyboard:
laListLibraryBlocks
Icon:
left click
Toolbar:
Menu:
Lands Design > Documentation > Listing > Urban furniture list

The laListLibraryBlocks command inserts a 2D table displaying the information of the existing urban furniture blocks in the model.

The features of urban furniture blocks lists are the same as the rest of listing object tools.

  • Insert a urban library block list
  • Insert options and parameters
  • Edit options
  • Graphical display


Urban furniture list.

Urban furniture fields

These are the default fields for the urban furniture list object, available under the Element (Urban furniture) list tab:

  • Block icon: you can edit the scale value to control the icon size.
  • Index: block identifier (ID). Lands assigns an ID value to blocks by default.
  • Code
  • Name
  • Amount: number of block items in the model.
