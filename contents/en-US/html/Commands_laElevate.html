Elevate

Where can I find this command?
Keyboard:
laElevate
Icon:
left click
Menu:
Lands Design > Edit > Adjust > Elevate
Edit Panel:
Select a Lands element > Edit area in Edit Panel > Adjust > Elevation

The laElevate command moves any kind of object vertically.

Steps:

  1. Select the object to elevate.
  2. Specify an offset distance. You can either enter this value numerically or by measuring the distance with two clicks in the model.
