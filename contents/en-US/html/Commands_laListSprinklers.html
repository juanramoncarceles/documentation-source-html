Sprinklers List

Where can I find this command?
Keyboard:
laListSprinklers
Icon:
left click
Toolbar:
Menu:
Lands Design > Documentation > Listing > Sprinkler list

The laListSprinklers command inserts a 2D table displaying the information of the existing sprinklers in the model.

The features of sprinkler lists are the same as the rest of listing object tools.

  • Insert a sprinkler list
  • Insert options and parameters
  • Edit options
  • Graphical display


Sprinkler list.

Sprinkler list fields

These are the default fields for the sprinkler list object, available under the Element (Sprinkler) list tab:

  • Sprinkler icon: sprinkler symbol displayed. You can edit the scale value to control the icon size.
  • Index: sprinkler identifier (ID). Lands assigns an ID value to sprinklers by default.
  • Type: sprinkler type.
  • Radius
  • Sweep angle
  • Description: text displayed in the sprinkler description field.
  • Amount: number of sprinklers in the model. The list will display the sprinklers individually if they have different properties (Type, Sweep angle, Radius, etc.) and these properties are enabled in the list.
