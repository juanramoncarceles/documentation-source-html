Reverse Curve

Where can I find this command?
Keyboard:
laReverseCurve
Menu:
Lands Design > Edit > Reverse Curve
Edit Panel:
Select a Lands element > Edit area in Edit panel > Reverse Curve

The laReverseCurve command flips the direction of objects generated from path curves. The start of the path curve will become the end and viceversa, and Lands elements along it will be redistributed accordingly.

This command is available from the Edit area in Edit panel, for the following objects:

  • Plant row
  • Dwarf Wall - Hedge
  • Fence
  • Row
  • Path
  • Stair

Steps:

  1. Select the object to reverse.
