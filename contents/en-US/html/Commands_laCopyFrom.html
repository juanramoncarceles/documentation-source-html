Copy properties from another object

Where can I find this command?
Keyboard:
laCopyFrom
Menu:
Lands Design > Edit > Copy properties from another object
Edit Panel:
Select a Lands element > Edit area in Edit Panel > Copy properties from another object

The laCopyFrom command changes the properties of a selected Lands object to match those of another Lands object.

This command is available from the Edit area in Edit panel for the following objects:

  • Plant row
  • Forest
  • Shrub
  • Dwarf Wall - Hedge
  • Fence
  • Row
  • Path
  • Stair
  • Zone

Steps:

  1. Select the object to edit.
  2. Select the object from which you want to copy parameters.
