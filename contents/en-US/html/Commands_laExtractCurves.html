Extract Curves

Where can I find this command?
Keyboard:
laExtractCurves
Menu:
Lands Design > Edit > Extract Curves
Edit Panel:
Select a Lands element > Edit area in Edit Panel > Extract curves

The laExtractCurves command obtains the curves used to generate Lands elements.

This command is available from the Edit area in Edit panel, for the following objects:

  • Plant row
  • Forest
  • Shrub
  • Dwarf Wall - Hedge
  • Fence
  • Row
  • Path
  • Stair
  • Zone

Steps:

  1. Select the object to extract the curves from.
