Modifier

Où puis-je trouver cette commande ?
Clavier :
laModifier
Icône :
clic gauche [clic long]
Barre d'outils :
Menu :
Lands Design > Modifier > Modifier...
Raccourci :
Double-clic sur l'objet à modifier

La commande laModifier ouvre la boîte de dialogue Explorateur de propriétés pour une sélection d'objets de Lands Design et vous permet de les modifier. Cette boîte de dialogue est similaire à la boîte de dialogue Propriétés mais elle affiche la liste des objets sélectionnés.

Lorsque aucun objet n'est sélectionné, la boîte de dialogue des propriétés affiche les Propriétés du document.

Lorsque plusieurs objets de Lands Design sont sélectionnés, cette boîte de dialogue affiche la liste des entités sélectionnées, regroupées par type d'élément (plante, terrain, forêt, arbuste, etc.).


Boîte de dialogue Propriétés pour différents éléments de Lands Design.
