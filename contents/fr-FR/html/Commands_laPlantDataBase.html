Lands Design présente une grande base de données de plantes, basée sur Visualplant, qui peut être utilisée par le professionnel de l'aménagement paysager pour insérer des espèces de plantes dans le dessin. Elle peut être très utile également comme référence, pour faire des recherches ou pour imprimer les données botaniques de différentes espèces.

Comment s'organise cette boîte de dialogue ?

La boîte de dialogue de la base de données de plantes se divise en plusieurs zones :


Boîte de dialogue Base de données de plantes.


  • 1. Zone d'outils
  • 2. Filtre
  • 3. Liste des espèces de plantes
  • 4. Zone d'affichage

Ligne d'outils


Ligne d'outils dans la base de données de plantes.

Nouvelle espèce : ouvre une boîte de dialogue des propriétés d'une espèce de plantes vide dans laquelle vous pouvez définir les caractéristiques et paramètres de la nouvelle espèce de végétation.

Supprimer une espèce : supprime une espèce de plante sélectionnée de la liste.

Propriétés : ouvre la boîte de dialogue des propriétés de l'espèce de végétation sélectionnée.

Aperçu de la fiche : ouvre une fenêtre dans le navigateur avec des informations sur l'espèce de plante sélectionnée.

Images Google : active une recherche Google de l'espèce de plantes sélectionnée et affiche les images trouvées dans la zone d'affichage, remplaçant ainsi les images et illustrations existantes. Une connexion à Internet active est nécessaire pour activer cette option. Les images trouvées sont automatiquement enregistrées dans un dossier appelé "Google Plant Images" disponible dans les fichiers publics de Windows : (C:\Utilisateurs\Public\Documents\Lands).

Filtre


Zone des filtres de la base de données de plantes.

Cette zone affiche différentes options pour filtrer la recherche d'espèces de plantes :

Par groupes d'espèces

Un menu déroulant affiche trois options : Favoris , Avec rendu pour afficher les espèces possédant un fichier spécifique de rendu (.arPlantX) et Tout.

Par nom ou nom commun, en effectuant une recherche

Une case de recherche est affichée à côté du menu déroulant des groupes. Un texte peut être entré dans cette case pour filtrer les plantes, afin d'afficher uniquement celles dont le nom ou le nom commun contiennent ce texte.

Caractéristiques

La case Filtres active la recherche d'espèces de plantes selon leur morphologie et caractéristiques.

Zones climatiques

Une carte du monde et une liste des différentes zones climatiques classées par couleur. Cette partie de la zone des filtres peut être masquée de la recherche avec la flèche à droite.

Liste des espèces de plantes


Liste d'espèces de plantes dans la base de données de plantes.

Cette zone montre la liste des espèces de plantes en fonction des critères de recherche. La liste peut être classée par ordre alphabétique ou numérique en cliquant sur le titre de la catégorie correspondante :

  • Favoris : indique si une espèce est marquée en tant que favori. Cliquez sur l'icône pour ajouter ou retirer une espèce du groupe Favoris.
    • Favori
    • Non favori
  • Nom
  • Nom commun
  • Type de plante
  • Type de feuille
  • Hauteur maximum
  • Forme

Affichage


Zone d'affichage dans la base de données des plantes.

Cette zone est divisée en deux parties :

  • Partie supérieure : affiche des images ou illustrations de l'espèce de plante sélectionnée en fonction de l'activation ou non de l'option Images Google.
  • Partie inférieure : montre les différents affichages 2D (dessin de la cime et symbole d'implantation) et 3D (conceptuel, détaillé et réaliste) de cette espèce de plante dans le modèle. Si vous cliquez sur un des affichages 2D/3D, la boîte de dialogue de l'espèce de plante s'ouvre sur l'onglet Affichage.

Propriétés de l'espèce de plante


Boîte de dialogue des propriétés de l'espèce de plante, onglet Informations sur l'espèce.


La boîte de dialogue Propriétés de l'espèce de plante montre toutes les informations sur chaque espèce de plante, organisées dans différents onglets :

Informations sur l'espèce

Cet onglet affiche les informations de base sur l'espèce de plante : origine, famille, zones climatiques, conditions requises, utilisation et commentaires.

Morphologie

Les informations suivantes peuvent être définies pour chaque espèce de plante :

  • Morphologie : Type de plante, Type de feuille, Forme, Système racinaire, Croissance et Hauteur maximum.
  • Saison : Saison de floraison, Couleur de floraison, Saison de production des fruits et Couleur des fruits.
  • Besoins en eau
  • Description de la feuille
  • Notes de l'utilisateur

Caractéristiques

Les informations suivantes peuvent être définies pour chaque espèce de plante :

  • Caractéristiques : Type de plante, Type de feuille, Forme, Système racinaire, Croissance et Hauteur maximum.
  • Type de sol
  • Niveaux de résistance à la pollution, à l'environnement maritime, aux inondations, au vent, aux fléaux et au soleil.
  • Description de la feuille
  • Notes de l'utilisateur

Affichage

Cet onglet indique les différents affichages 2D et 3D pour les dessins, la forme conceptuelle et la forme détaillée de la plante.
Dessin 2D de la cime Forme 3D conceptuelle Forme 3D détaillée

Rendu

Cet onglet affiche le fichier de plante rendu (fichier.ArPlantX) assigné à cette espèce. Cliquez sur le bouton Modifier pour chercher un autre fichier .ArPlantX dans la bibliothèque de Lands Design ou cliquez sur le bouton Éditeur de plantes pour ouvrir et modifier le fichier nXt actuel.


Boîte de dialogue des propriétés de l'espèce de plante, onglet Rendu.
