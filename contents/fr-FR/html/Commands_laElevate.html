Élever

Où puis-je trouver cette commande ?
Clavier :
laElever
Icône :
clic gauche
Barre d'outils :
Menu :
Lands Design > Modifier > Ajuster > Élever
Panneau d'édition :
Sélectionner un élément de Lands > Zone d'édition du panneau d'édition > Ajuster > Élévation

La commande laElever déplace n'importe quel type d'objet verticalement.

Étapes :

  1. Sélectionnez l'objet à élever.
  2. Indiquez la distance de décalage. Vous pouvez entrer une valeur numérique ou indiquer la distance avec deux clics dans le modèle.
