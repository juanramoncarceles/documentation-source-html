Insérer un mur :

Lorsque vous lancez la commande laMur, la boîte de dialogue d'insertion d'un mur apparaît. Cette boîte de dialogue présente les options de base du mur dans deux onglets.

  • Mur : paramètres du mur.
  • Matériau : paramètres de matériau pour le mur.

Étapes :

  1. Sélectionnez les paramètres du mur. Cliquez sur Accepter pour fermer la boîte de dialogue
  2. Sélectionnez une courbe du modèle pour définir le chemin du mur ou sélectionnez une option dans la ligne de commande.
    • Polyligne : le mur sera créé comme si vous dessiniez une polyligne.
    • Spline : le mur sera créé comme si vous dessiniez une spline.
    • Cercle : le mur sera créé comme si vous dessiniez un cercle. Le premier clic définit le centre du cercle et le deuxième indique le rayon.
    • Arc : le mur sera créé comme si vous dessiniez un arc. Le premier clic définit le point d'insertion. Le deuxième indique le point central de l'arc. Le troisième et le dernier déterminent les extrémités de l'arc.
    • Rectangle : le mur sera créé comme si vous dessiniez un rectangle. Le premier clic définit un sommet du rectangle et le deuxième le sommet opposé.
  3. Appuyez sur Entrée, Échap ou cliquez avec le bouton droit pour terminer la commande.

Points de contrôle

Les murs ont les mêmes points de contrôle que les courbes à partir desquelles ils ont été créés.

Ils disposent d'une flèche de contrôle du déplacement afin de déplacer tout l'objet dans son ensemble.

Les points de contrôle des murs sont automatiquement activés lorsqu'un mur est sélectionné. Pour activer les points de contrôle sur un ou plusieurs murs, sélectionnez-les et lancez la commande ActiverPoints (ou appuyez sur F10). Pour désactiver les points de contrôle sur un mur, appuyez sur Échap ou lancez la commande DésactiverPoints.

Paramètres et options d'insertion

Les paramètres et options d'insertion des murs se trouvent dans différents onglets de plusieurs boîtes de dialogue.

  • Boîte de dialogue d'insertion d'un mur (uniquement disponible lors de l'insertion de nouveaux murs).
  • Section propriétés des murs, dans le panneau d'édition de Lands Design.
  • Boîte de dialogue des propriétés objet d'un mur.

Général

Attributs généraux du mur : Nom, Couleur, Code, et Mots clé.

Mur

  • Épaisseur : épaisseur constante tout le long du mur.
  • Hauteur : hauteur constante tout le long du mur.
  • Aspect lisse : précision avec laquelle le mur est subdivisé le long de la courbe de l'axe afin de s'adapter à la forme. Ce paramètre n'est important que lorsque le mur a été créé à partir d'une courbe présentant une courbure.
  • Alignement : marque la position de l'axe par rapport à la structure. Il existe trois alignements possibles :
    • Gauche : la courbe guide se trouve sur la gauche du mur.
    • Milieu : le mur sera construit en répartissant la moitié de son épaisseur de chaque côté de la courbe guide.
    • Droite : la courbe guide se trouve sur la droite du mur.

Matériau

Cet onglet permet de changer l'image de texture assignée à cet élément et de définir la taille, la réflectivité et la transparence de l'image.

Options d'édition

Options d'édition des murs, disponibles dans la zone d'édition du panneau d'édition :

  • Actualiser
  • Convertir en 2D. Affiche l'objet avec un affichage 2D.
  • Convertir en 3D. Affiche l'objet avec un affichage 3D.
  • Afficher en 2D et 3D : Active les représentations 2D et 3D de l'objet simultanément.
  • Copier les propriétés d'un autre objet
  • Copier sur une autre courbe
  • Inverser la courbe
  • Extraire une courbe
  • Ajuster
    • Élévation
    • À l'élévation zéro
    • Au terrain
  • Ne pas ajuster au terrain

Affichage des murs :

Les murs peuvent être affichés simultanément en 2D et 3D dans le dessin.
