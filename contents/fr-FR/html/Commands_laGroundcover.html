Les couvre-sol de Lands Design sont des zones de végétation remplies avec une espèce de plante selon des paramètres de densité et représentées sous forme de hachures pour l'affichage en 2D.

Insérer un couvre-sol :

Lorsque vous lancez la commande laCouvreSol, la boîte de dialogue d'insertion d'un couvre-sol apparaît. Cette boîte de dialogue présente les options de base du couvre-sol dans un onglet unique.

Étapes :

  1. Sélectionnez une espèce de couvre-sol dans la liste. Cliquez sur le bouton Parcourir... pour en sélectionner une nouvelle dans la base de données de plantes.
  2. Définissez les autres paramètres d'insertion (mode de présentation, motif de hachures et fichier de plante de nXt) puis cliquez sur Accepter pour fermer la boîte de dialogue.
  3. Sélectionnez une courbe du modèle pour définir la frontière du couvre-sol ou sélectionnez une option dans la ligne de commande.
    • Polyligne : le couvre-sol sera créé comme si vous dessiniez une polyligne.
    • Spline : le couvre-sol sera créé comme si vous dessiniez une spline.
    • Cercle : le couvre-sol sera créé comme si vous dessiniez un cercle. Le premier clic définit le centre du cercle et le deuxième indique le rayon.
    • Arc : le couvre-sol sera créé comme si vous dessiniez un arc. Le premier clic définit le point d'insertion. Le deuxième indique le point central de l'arc. Le troisième et le dernier déterminent les extrémités de l'arc.
    • Rectangle : le couvre-sol sera créé comme si vous dessiniez un rectangle. Le premier clic définit un sommet du rectangle et le deuxième le sommet opposé.
  4. Appuyez sur Entrée, Échap ou cliquez avec le bouton droit pour terminer la commande.

Points de contrôle

Les couvre-sol ont les mêmes points de contrôle que les courbes à partir desquelles ils ont été créés.

Ils disposent d'une flèche de contrôle du déplacement afin de déplacer tout le couvre-sol dans son ensemble.

Les points de contrôle des couvre-sol sont automatiquement activés lorsqu'un couvre-sol est sélectionné. Pour activer les points de contrôle sur un ou plusieurs couvre-sol, sélectionnez-les et lancez la commande ActiverPoints (ou appuyez sur F10). Pour désactiver les points de contrôle, appuyez sur Échap ou lancez la commande DésactiverPoints.

Options et paramètres d'insertion

Les paramètres et options d'insertion des couvre-sols se trouvent dans différents onglets de plusieurs boîtes de dialogue.

  • Boîte de dialogue d'insertion d'une plante (uniquement disponible lors de l'insertion de nouvelles plantes).
  • Section propriétés des plantes, dans le panneau d'édition de Lands Design.
  • Boîte de dialogue des propriétés objet d'une plante.

Général

Attributs généraux des couvre-sol : Nom, Couleur, Code, et Mots clé.

Couvre-sol

  • Espèce : Nom de l'espèce de plante ou de couvre-sol sélectionnée. La case affiche la liste des différentes espèces de plantes qui ont été insérées dans le modèle actuel. Vous pouvez sélectionner une espèce dans la liste ou en choisir une autre dans la base de données de plantes après avoir cliqué sur le bouton Parcourir....
  • Mode de présentation : trois modes de présentation sont disponibles mais vous pouvez en créer de nouveaux.
    • Racine nue
    • Motte
    • Godet
  • Densité : indique la quantité de spécimens de végétation en unités/m2 à l'intérieur de la courbe frontière.
  • Motif de hachures : paramètres pour la représentation du couvre-sol en 2D.
  • Fichier de plante de nXt : les couvre-sols utilisent des fichiers de couvre-sol (".ArGroundcoverX") pour leur représentation 3D.

Options d'édition

Options d'édition des couvre-sol, disponibles dans la zone d'édition du panneau d'édition :

  • Actualiser
  • Copier les propriétés d'un autre objet
  • Randomiser
  • Extraire des courbes
  • Ajuster
    • Élévation
    • À l'élévation zéro
    • Au terrain
  • Ne pas ajuster au terrain
  • Couvre-sol
    • Étiquette (Cette commande apparaît uniquement lorsque le couvre-sol n'ont pas encore d'étiquette).
    • Étiquette d'identification (Cette commande apparaît uniquement lorsqu'une étiquette de plante est définie pour l'espèce composant le couvre-sol)
    • Insérer une photo
    • Ajouter un trou : soustrait une frontière dans un couvre-sol

Affichage des couvre-sol :

Lands Design propose différents affichages simultanés en 2D et 3D des couvre-sol dans le dessin, afin de pouvoir manipuler les plantes à partir de trois niveaux conceptuels différents. L'affichage des couvre-sol peut être modifié dans les propriétés du document.
