Panneau d'édition

Où puis-je trouver cette commande ?

Icône :

Barre d'outils : Outils de Lands
Menu :
Lands Design > Panneau d'édition

Le panneau d'édition de Lands Design est ancrable et permet d'accéder à toutes les options d'édition en temps réel pour tous les objets du modèle.

Dans le panneau d'édition, vous pouvez :

  • Modifier les propriétés des objets sélectionnés
  • Lancer les fonctions proposées par le menu contextuel pour la sélection actuelle
  • Changer le mode de présentation des plantes 3D
  • Appliquer des définitions de calques
  • Appliquer des filtres sur le dessin

Le panneau d'édition est divisé en quatre sections ou zones :

  1. Ligne d'outils
  2. Liste de sélections d'objets
  3. Zone d'édition
  4. Zone des propriétés de l'objet


Panneau d'édition de Lands Design (lorsque aucun objet n'est sélectionné).

1. Ligne d'outils

La ligne d'outils du panneau d'édition présente les options suivantes.

  • Bouton d'affichage 2D et 3D : ils sont utilisés pour passer de la représentation 2D à la 3D et vice-versa pour tous les objets de Lands Design dans le modèle. Cette opération s'applique à toutes les fenêtres de travail.
  • Bouton de sélection : affiche une liste de tous les types d'éléments de Lands Design, où vous pouvez sélectionner les éléments d'un même type.
  • Ajuster au terrain : permet de placer automatiquement les objets sur la surface d'un terrain lors de leur insertion.
  • Actualiser : actualise automatiquement toute modification réalisée sur un objet de Lands Design.

2. Liste de sélection

Cette zone affiche les objets sélectionnés dans le modèle.

Lorsque aucun objet n'est sélectionné, la liste affiche les différentes espèces de végétation et les blocs de mobilier urbain existants dans le modèle ainsi que le nom de chaque élément.

En sélectionnant une espèce de plante dans la liste, tous les spécimens de cette espèce seront sélectionnés dans le modèle.

En double cliquant sur l'icône près de l'espèce, vous pourrez montrer ou masquer toutes les plantes de la même espèce.

  • Les spécimens de cette espèce sont visibles.
  • Les spécimens de cette espèce sont cachés.


Zone de la liste de sélection dans le panneau d'édition.

3. Zone d'édition

Cette zone montre les commandes d'édition disponibles pour la sélection actuelle. (Ces commandes se trouvent aussi dans le menu contextuel de la sélection actuelle.)


Zone d'édition du panneau d'édition (pour une forêt).

4. Zone des propriétés

Cette zone est organisée en onglets qui montrent les différentes propriétés et options pour chaque type d'objet. (Ces propriétés sont aussi disponibles dans la boîte de dialogue des propriétés d'objet.)

Quand aucun objet n'est sélectionné, cette zone affiche les Propriétés du document.

Lorsque l'option Appliquer chaque changement est activée, toutes les modifications appliquées sur l'objet édité sont actualisées automatiquement.


Zone des propriétés du panneau d'édition (pour une sélection de plantes).
