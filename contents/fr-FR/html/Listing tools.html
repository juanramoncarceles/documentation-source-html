Commandes de création et édition de listes


Les commande de création de listes créent des tableaux 2D avec les informations sur les éléments de Lands Design insérés dans le modèle.

  • Espèces de plantes
  • Mobilier urbain
  • Terrains
  • Arroseurs
  • Canalisations
  • Zones

Les listes peuvent être exportées vers Microsoft Excel ou au format HTML grâce à la commande laExporterListe.

Insérer une liste :

Étapes :

  1. Lancez une des commandes de création de liste.
  2. Indiquez un point dans le modèle où insérer la liste.

La liste contient les informations de tous les éléments de ce type se trouvant dans le modèle et elle est automatiquement mise à jour lorsque des éléments sont ajoutés ou supprimés dans le modèle.


Paramètres et options d'insertion

Les paramètres des listes se trouvent dans différents onglets de plusieurs boîtes de dialogue :
  • Section propriétés des listes dans le panneau d'édition de Lands Design.
  • Boîte de dialogue des propriétés d'objet pour une liste.

Général

Attributs généraux des listes : Nom, Couleur, Calque et Transparence (pour le mode d'affichage Conceptuel uniquement).

Tableau

Cet onglet définir le format du texte et les attributs du titre, de l'en-tête et des données de la liste. Vous pouvez aussi définir les marges de la cellule et les propriétés de la bordure de la liste.


Onglet Tableau.


Bordures

Cet onglet définir les bordures du tableau.


Onglet Bordures.


Liste des éléments

Cet onglet permet de modifier les champs à afficher dans la liste et comment ils apparaissent dans l'en-tête. Lands Design assigne un nom par défaut à tous les champs. Ces champs sont différents pour chaque élément de Lands.

  • Champs à montrer : paramètres disponibles pour l'élément. Cliquez dans les cases pour montrer ou masquer chaque élément.
  • Nom du champ : cliquez dans ce champs pour changer le nom de l'en-tête.
  • Trier par : cliquez dans la ligne du paramètre que vous voulez utiliser pour ordonner les valeurs du tableau sous Trier par. Cliquez à nouveau pour passer du mode Ascendant au mode Descendant et vice versa.

Flèches : permettent de changer l'ordre dans lequel les champs sont affichés dans le tableau.

Échelle de l'icône : affiche la représentation 2D de l'objet. Vous pouvez modifier la valeur d'échelle pour contrôler la taille de l'icône.

Lister les objets : vous pouvez décider si les éléments sur des calques désactivés ou cachés sont pris en compte dans la liste ou non.


Onglet de la liste des éléments pour une liste de plantes.


Options d'édition

Options d'édition des objets de liste, disponibles dans la zone d'édition du panneau d'édition :

  • Actualiser
  • Liste
    • Exporter
    • Voir les onglets : (uniquement pour les listes de plante). Ouvre la base de données de plantes avec les plantes incluses dans la liste.

Affichage des listes :

Une liste est un objet 2D et elle est toujours affichée en 2D, quel que soit le paramètre du mode d'affichage 2D ou 3D dans le panneau d'édition de Lands.
