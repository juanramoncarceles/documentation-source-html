Options de Lands Design

Vous pouvez modifier certaines options et préférences pour adapter Lands à vos besoins.

Ces paramètres concernent tous les documents de Lands lorsque le programme est lancé.

Pour ouvrir la boîte de dialogue Options de Lands Design, ouvrez le menu déroulant Lands Design > Options...

La boîte de dialogue des Options de Lands Design s'ouvrira et affichera plusieurs onglets. Chaque onglet affiche une section avec plusieurs paramètres :

  • Général
  • Calques
  • Sources de données


Options de Lands Design, onglet Général

Général

L'onglet Général est divisé en trois parties :

  • Langue : Vous pouvez sélectionner la langue de l'interface de Lands Design parmi celles que Rhino a installé sur l'ordinateur. Lands Design est disponible dans 6 langues : français, anglais, allemand, italien, espagnol et chinois (simplifié). Redémarrez Lands Design après avoir changé de langue.
  • Modifier : Certaines modifications concernant l'édition d'objets peuvent être appliquées :
    • Mise à jour automatique : Active ou désactive l'actualisation automatique des objets de Lands Design dans le dessin.
    • Ouvrir le Panneau d'édition au démarrage : vous pouvez choisir d'afficher ou non le panneau d'édition à chaque démarrage de Lands Design.
    • Valeurs maximales pour modifier les objets : définit le nombre maximum d'unités dans les files, les forêts et les matrices ainsi que le nombre maximum de points dans la grille du terrain.
  • Notifications et messages : Choisissez si vous voulez voir la boîte de dialogue de bienvenue, l'écran de présentation et si vous voulez que Lands Design vous demande quel système d'unités vous voulez utiliser au démarrage.

Calques

Cet onglet définit le nom et la couleur des calques que Lands Design créera pour chaque type d'objet lors de leur insertion dans le modèle.

Lorsque que la case à côté d'un type d'objet est désactivée, l'élément de Lands est créé sur le calque indiqué dans les Propriétés du document (Options de Rhino > Propriétés du document > Lands Design > Calques). Lorsque le calque est défini sur Aucun, l'objet est créé sur le calque actuel.

Les associations objet-calque définies dans le dessin actuel peuvent être vérifiées dans l'onglet Calques des Propriétés du document en bas du panneau d'édition.


Boîte de dialogue Options de Lands Design, onglet Général

Origines des données

Blocs de mobilier urbain

Cet onglet vous aide à gérer les répertoires où sont enregistrés les blocs de Lands et autres blocs.

Cliquez sur le bouton Nouveau... pour ouvrir la boîte de dialogue Origine des données et créer un nouveau répertoire. Choisissez le nom et l'emplacement du répertoire.

Cliquez sur Modifier... pour modifier le nom et l'emplacement du répertoire et sur Supprimer pour supprimer un emplacement de la liste.

Dossier de contenu centralisé

Cette option permet de sélectionner un dossier partagé sur un réseau afin de partager du contenu (textures de plantes, photos, modifications dans la base de données) avec d'autres membres de votre équipe. Si ce chemin est vide, Lands créera le contenu localement pour l'ordinateur actuel.


Boîte de dialogue Options de Lands Design, onglet Origine des données



Boîte de dialogue Origine des données

Rendu

Cet onglet permet de définir un dossier temporaire pour enregistrer les fichiers de texture afin que d'autres ordinateurs sur le réseau puissent accéder à ceux-ci.

Options de Lands Design (dans la boîte de dialogue Options de Rhino)

Les options de Lands Design sont également disponibles dans la boîte de dialogue Options de Rhino (Menu Outils > Options...).
