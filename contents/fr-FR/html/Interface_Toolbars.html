Barres d'outils

Les barres d'outils de Lands Design fonctionnent exactement comme celles de Rhino :

  • Elles peuvent être déplacées et laissées flottantes dans l'écran ou être ancrées sur le bord du modèle.
  • Elles peuvent aussi être modifiées en ajoutant ou en supprimant des boutons.
  • Lorsque le curseur est placé sur un bouton, une légende affiche le nom des commandes qui peuvent être lancées avec le bouton de droite et le bouton de gauche.
  • Certains boutons (avec un triangle en bas à droite) possèdent une barre d'outils liée.

Cliquez sur l'icône et maintenez enfoncé le bouton de gauche de la souris pour afficher la barre d'outils liée à un bouton.

Déplacez le curseur de la souris sur une icône pour afficher son info-bulle :


Info-bulle de l'icône Plante (clic gauche)

Les barres d'outils de Lands Design sont organisées par défaut dans 3 onglets :

  • Objets de Lands
  • Documentation de Lands
  • Outils de Lands
Il est possible de charger la barre d'outils Lands Design qui comprend toutes les commandes dans un seul onglet.

    REMARQUE : Si les barres d'outils ne sont pas automatiquement affichées après l'installation de Lands Design, elles peuvent être chargées manuellement en utilisant la même méthode que pour charger les barres d'outils de Rhino : Tapez BarreOutils dans la ligne de commande et ouvrez le fichier Lands Design.rui qui se trouve dans le répertoire suivant :

    • Rhino 6 (64 bits) : (C:\Utilisateurs\"Nom d'utilisateur"\AppData\Roaming\AsuniCAD\Lands\Rhinoceros\64\bit)\UI).
