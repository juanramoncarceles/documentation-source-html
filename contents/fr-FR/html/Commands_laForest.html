Insérer une forêt :

Lorsque vous lancez la commande laForêt, la boîte de dialogue d'insertion d'une forêt apparaît. Cette boîte de dialogue présente les options de base de la forêt dans quatre onglets :

  • Forêt : paramètres de la forêt.
  • Plante : paramètres des spécimens de l'espèce de plante dans la forêt.
  • Format : format de l'espèce de plante dans la forêt.
  • Rendu : paramètres qui définissent comment les plantes de la forêt apparaîtront dans les vues rendues.

Étapes :

  1. Sélectionnez une espèce de plante dans la liste, sous l'onglet Plante. Cliquez sur le bouton Parcourir... pour en sélectionner une nouvelle dans la base de données de plantes. Définissez les autres paramètres de la plante.
  2. Ouvrez l'onglet Forêt et définissez les paramètres de la forêt. Cliquez sur Accepter pour fermer la boîte de dialogue.
  3. Choisissez une option dans la ligne de commande pour créer la forêt :
    • Limites : sélectionnez une Courbe ou plusieurs courbes qui seront utilisées pour définir les limites de la forêt. Les courbes doivent être fermées.
    • PremierPoint : cliquez dans le modèle pour définir la courbe de frontière de la forêt comme si vous dessiniez une polyligne. Cliquez avec le bouton droit pour terminer la commande. Si le dernier point d'insertion ne correspond pas au premier point, un segment sera ajouté pour fermer la frontière.
    • PointIntérieur : (option non disponible dans la version WIP de Lands Design).
  4. Appuyez sur Entrée, Échap ou cliquez avec le bouton droit pour terminer la commande.

Les paramètres de la forêt peuvent être modifiés lors de l'insertion et après, à partir du panneau d'édition de Lands Design, dans la section Propriétés, sous l'onglet Forêt.

Points de contrôle

Les forêts ont les mêmes points de contrôle que les courbes à partir desquelles elles ont été créées.

Elles disposent d'une flèche de contrôle du déplacement afin de déplacer toute la forêt dans son ensemble.

Les points de contrôle des forêts sont automatiquement activés lorsqu'une forêt est sélectionnée. Pour activer les points de contrôle sur une ou plusieurs forêts, sélectionnez-les et lancez la commande ActiverPoints (ou appuyez sur F10). Pour désactiver les points de contrôle, appuyez sur Échap ou lancez la commande DésactiverPoints.

Options et paramètres d'insertion

Les paramètres et options d'insertion des forêts se trouvent dans différents onglets de plusieurs boîtes de dialogue.

  • La boîte de dialogue d'insertion d'une forêt (uniquement disponible lors de l'insertion de nouvelles forêts).
  • Section propriétés des forêts, dans le panneau d'édition de Lands Design.
  • Boîte de dialogue des propriétés objet d'une forêt.

Général

Attributs généraux des forêts : Nom, Couleur, Code, et Mots clé.

Forêt

Deux options sont disponibles pour distribuer les plantes dans des forêts :

  • Nombre d'unités : distribue un nombre spécifique de plantes aléatoirement à l'intérieur de la courbe de frontière. Vous pouvez indiquer la Distance minimale entre les éléments. Vous pouvez modifier la distribution aléatoire dans la section d'édition du panneau de d'édition : Forêt > Redistribuer
  • Matrice : organise les spécimens de plante sur des lignes et colonnes.
    • Espacement : carré, rectangulaire, triangulaire équilatéral et triangulaire isocèle.
    • Rotation : toute la forêt tourne autour de son point central.
    • Dispersion : déplace les plantes des points de la matrice.

Les forêts peuvent être composées de plusieurs espèces de plantes :

  • Nb d'éléments : tapez le nombre de spécimens de plantes. La liste des différentes espèces apparaîtra dans la Liste de sélection d'objets du Panneau d'édition de Lands, d'où vous pouvez les sélectionner pour modifier leurs paramètres.
  • Distribution : options pour distribuer les espèces de plantes lorsqu'il y a plusieurs types.

Unités insérées : affiche le nombre exact d'unités de plantes dans la forêt. Lorsque la forêt est définie avec le mode Nombre d'unités, la quantité d'unités de plantes dépend de la distance minimale entre les éléments.

Plante

  • Espèce : Nom de l'espèce de plante sélectionnée. La case affiche la liste des différentes espèces de plantes qui ont été insérées dans le modèle actuel. Vous pouvez sélectionner une espèce dans la liste ou en choisir une autre dans la base de données de plantes après avoir cliqué sur le bouton Parcourir....
  • Cotes sur le dessin :
    • Diamètre du sommet
    • Hauteur approximative
    • Age
  • Variation naturelle : lorsque cette option est activée, elle définit une variation aléatoire de hauteur sur les différentes plantes dans la forêt. La valeur maximum (100 %) crée des spécimens de plantes dont la taille varie entre 50 et 150 %.
  • Rotation aléatoire : lorsque cette option est activée (par défaut), elle définit une rotation aléatoire des plantes. Lorsqu'elle est désactivée, vous pouvez définir un angle de rotation spécifique pour chaque plante. L'angle de rotation est mesuré à partir de l'axe vertical de la plante.

Les valeurs de hauteur minimum et maximum des plantes (âge) sont définies pour chaque espèce de plante et peuvent être modifiées dans la boîte de dialogue des propriétés de l'espèce de plante, disponible dans la base de données des plantes.

Format

  • Calibre : dimensions de l'écorce de l'espèce de plantes. Des valeurs entre 10 et 130 mm sont disponibles par défaut mais vous pouvez taper n'importe quelle valeur.
  • Mode de présentation : trois modes de présentation sont disponibles mais vous pouvez en créer de nouveaux.
    • Racine nue
    • Motte
    • Godet
    •

    Rendu

    Cet onglet définit comment les espèces de plantes des forêts seront affichées dans une vue rendue. Il n'est visible que lorsque l'affichage 3D des plantes est défini sur Réaliste dans les propriétés du document.

    • Saison : Lorsque la case Ignorer la saison générale est cochée, l'utilisateur peut choisir la saison qui sera utilisée pour rendre les plantes, en sélectionnant une des 7 valeurs disponibles : Début du printemps, Printemps, Fin du printemps, Été, Fin de l'été, Automne, Hiver...
    • Densité du feuillage : Flamingo nXt générera plus ou moins de feuilles en fonction du niveau choisi.

    Options d'édition

    Options d'édition des forêts, disponibles dans la zone d'édition du panneau d'édition :

    • Actualiser
    • Dégrouper
    • Copier les propriétés d'un autre objet.
    • Ajuster
      • Élévation
      • À l'élévation zéro
      • Au terrain
    • Ne pas ajuster au terrain
    • Forêt
      • Redistribuer
      • Ajouter un trou : soustrait une frontière de la forêt.
      • Randomiser
    • Plante
      • Modifier des espèces
      • Étiquette (Cette commande apparaît uniquement lorsque les plantes de la forêt n'ont pas encore d'étiquette).
      • Étiquette d'identification (Cette commande apparaît uniquement lorsqu'une étiquette de plante est définie pour l'espèce composant la forêt.)
      • Insérer une photo

    Affichage des forêts :

    Les plantes des forêts sont affichées de la même façon que les espèces insérées individuellement. L'affichage graphique des plantes dans une forêt peut être modifié dans les propriétés du document.
