Définir le point de vue

Où puis-je trouver cette commande ?
Clavier :
laDéfinirPointDeVue
Icône :
clic gauche
Barre d'outils : Outils de Lands
Menu :
Lands Design > Définir un point de vue
Panneau d'édition :
Ligne d'outils > Définir le point de vue

La commande laDéfinirPointDeVue définit le point de vue de la caméra à partir de l'observateur et de la position de la cible.

Étapes

  1. Choisissez le point d'observation dans la fenêtre où vous voulez le définir.
  2. Choisissez un point sur la cible où l'observateur regardera.
