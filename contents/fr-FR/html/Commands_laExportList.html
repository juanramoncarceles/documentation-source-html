Exporter une liste

Clavier :
laExporterListe
Icône :
clic gauche
Barre d'outils :
Menu :
Lands Design > Documentation > Lister > Exporter une liste
Panneau d'édition :
Sélectionner une liste > Zone d'édition du panneau d'édition > Liste > Exporter

La commande laExporterListe exporte une liste existante de Lands Design vers un fichier .xls (Excel) ou .html.

Lorsque vous lancez la commande laExporterListe, la boîte de dialogue Exporter la liste apparaît. Cette boîte de dialogue possède les options suivantes :

  • Exporter : enregistre la liste vers le format .xls (Excel) ou .html.
  • Imprimer : imprime la liste.
  • Aperçu : affiche un aperçu de la liste dans une page web.
  • Liste à exporter : affiche les listes existantes dans le modèle. Choisir celle à exporter.
  • Liste : Contenu de la liste.
