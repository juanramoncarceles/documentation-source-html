Insérer une plante :

Lorsque vous lancez la commande laPlante, la boîte de dialogue d'insertion d'une plante apparaît. Cette boîte de dialogue présente les options de base de la plante dans deux onglets.

  • Plante : Espèce et Cotes sur le dessin.
  • Format : Calibre et Mode de présentation de l'espèce de plante
  • Rendu : définit comment la plante sera affichée dans une vue rendue.

Étapes :

  1. Sélectionnez une espèce de plante dans la liste, sous l'onglet Plante. Cliquez sur le bouton Parcourir... pour en sélectionner une nouvelle dans la base de données de plantes.
  2. Définissez les autres paramètres d'insertion (calibre, mode de présentation et cotes sur le dessin) puis cliquez sur Accepter pour fermer la boîte de dialogue.
  3. Indiquez le point d'insertion dans le modèle en cliquant. Un nouveau spécimen de l'espèce sera inséré à chaque clic. Vous pouvez modifier les paramètres de la plante lors de son insertion en utilisant le Panneau d'édition de Lands Design, dans la section Propriétés, sous l'onglet Pante.
  4. Appuyez sur Entrée, Échap ou cliquez avec le bouton droit pour terminer la commande.

Insérer des plantes à partir de points :

Lorsque vous lancez la commande laPlante, la boîte de dialogue d'insertion d'une plante apparaît. Cette boîte de dialogue présente les options de base de la plante.

Étapes :

  1. Sélectionnez une espèce de plante dans la liste, sous l'onglet Plante. Cliquez sur le bouton Parcourir... pour en sélectionner une nouvelle dans la base de données de plantes.
  2. Définissez les autres paramètres d'insertion (calibre, mode de présentation et cotes sur le dessin) puis cliquez sur Accepter pour fermer la boîte de dialogue.
  3. Sélectionnez l'option ÀPartirDePoints dans la ligne de commande.
  4. Sélectionnez des points dans le modèle où vous voulez placer les plantes.
  5. Appuyez sur Entrée, Échap ou cliquez avec le bouton droit pour terminer la commande.

Points de contrôle

Les plantes n'ont qu'un seul point de contrôle qui se trouve au point central inférieur de l'objet.

Le point de contrôle d'une plante est automatiquement activé lorsque celle-ci est sélectionnée. Pour activer les points de contrôle sur une ou plusieurs plantes, sélectionnez-les et lancez la commande ActiverPoints (ou appuyez sur F10). Pour désactiver les points de contrôle, appuyez sur Échap ou lancez la commande DésactiverPoints.

Paramètres et options d'insertion

Les paramètres et options d'insertion des plantes se trouvent dans différents onglets de plusieurs boîtes de dialogue.

  • Boîte de dialogue d'insertion d'une plante (uniquement disponible lors de l'insertion de nouvelles plantes).
  • Section propriétés des plantes, dans le panneau d'édition de Lands Design.
  • Boîte de dialogue des propriétés objet d'une plante.

Général

Attributs généraux pour la plante : Nom, Couleur, Code, et Mots clé.

Plante

  • Espèce : Nom de l'espèce de plante sélectionnée. La case affiche la liste des différentes espèces de plantes qui ont été insérées dans le modèle actuel. Vous pouvez sélectionner une espèce dans la liste ou en choisir une autre dans la base de données de plantes après avoir cliqué sur le bouton Parcourir....
  • Cotes sur le dessin :
    • Diamètre du sommet
    • Hauteur approximative
    • Age
  • Variation naturelle : lorsque cette option est activée, elle définit une variation aléatoire de hauteur sur les différentes plantes insérées ou sélectionnées. La valeur maximum (100 %) crée des spécimens de plantes dont la taille varie entre 50 et 150 %.
  • Rotation aléatoire : lorsque cette option est activée (par défaut), elle définit une rotation aléatoire des plantes. Lorsqu'elle est désactivée, vous pouvez définir un angle de rotation spécifique pour chaque plante. L'angle de rotation est mesuré à partir de l'axe vertical de la plante.

Les valeurs de hauteur minimum et maximum des plantes (âge) sont définies pour chaque espèce de plante et peuvent être modifiées dans la boîte de dialogue des propriétés de l'espèce de plante, disponible dans la base de données des plantes.

Format

  • Calibre : dimensions de l'écorce de l'espèce de plantes. Des valeurs entre 10 et 130 mm sont disponibles par défaut mais vous pouvez taper n'importe quelle valeur.
  • Mode de présentation : trois modes de présentation sont disponibles mais vous pouvez en créer de nouveaux.
    • Racine nue
    • Motte
    • Godet

Rendu

Cet onglet définit comment les plantes seront affichées dans une vue rendue. Il n'est visible que lorsque l'affichage 3D des plantes est défini sur Réaliste dans les propriétés du document.

  • Saison : Lorsque la case Ignorer la saison générale est cochée, l'utilisateur peut choisir la saison qui sera utilisée pour rendre les plantes, en sélectionnant une des 7 valeurs disponibles : Début du printemps, Printemps, Fin du printemps, Été, Fin de l'été, Automne, Hiver...
  • Densité du feuillage : Flamingo nXt générera plus ou moins de feuilles en fonction du niveau choisi.

Options d'édition

Options d'édition des plantes, disponibles dans la zone d'édition du panneau d'édition :

  • Actualiser
  • Copier les propriétés d'un autre objet.
  • Ajuster
    • Élévation
    • À l'élévation zéro
    • Au terrain
  • Ne pas ajuster au terrain
  • Plante
    • Modifier des espèces
    • Étiquette (cette commande apparaît uniquement lorsque la plante n'a pas encore d'étiquette)
    • Étiqueter en liant (cette commande apparaît uniquement lorsque la plante n'a pas encore d'étiquette)
    • Étiquette d'identification (cette commande apparaît uniquement lorsqu'une étiquette de plante est définie pour cette plante dans le modèle)
    • Insérer une photo

Affichage des plantes :

Lands Design propose différents affichages simultanés en 2D et 3D des plantes dans le dessin, afin de pouvoir les manipuler à partir de trois niveaux conceptuels différents. L'affichage des plantes peut être modifié dans les propriétés du document.
