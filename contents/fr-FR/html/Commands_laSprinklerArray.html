La matrice d'arroseurs insère une matrice d'arroseurs qui couvre une zone spécifique. La zone est définie avec une courbe fermée: polyligne, spline, cercle, etc.

Insère une matrice d'arroseurs

Lorsque vous lancez la commande laMatriceArroseurs, la boîte de dialogue d'insertion d'une matrice d'arroseurs apparaît. Cette boîte de dialogue présente les options de base dans deux onglets.

  • Arroseur
  • Matrice

Étapes :

  1. Sélectionnez les paramètres de l'arroseur dans l'onglet Arroseur et les paramètres de la matrice dans l'onglet Matrice. Cliquez sur Accepter pour fermer la boîte de dialogue.
  2. Sélectionnez une courbe du modèle pour définir la frontière de la matrice d'arroseurs ou sélectionnez une option dans la ligne de commande.
    • Polyligne : la matrice d'arroseurs sera créée comme si vous dessiniez une polyligne.
    • Spline : la matrice d'arroseurs sera créée comme si vous dessiniez une spline.
    • Cercle : la matrice d'arroseurs sera créée comme si vous dessiniez un cercle. Le premier clic définit le centre du cercle et le deuxième indique le rayon.
    • Arc : la matrice d'arroseurs sera créée comme si vous dessiniez un arc. Le premier clic définit le point d'insertion. Le deuxième indique le point central de l'arc. Le troisième et le dernier déterminent les extrémités de l'arc.
    • Rectangle : la matrice d'arroseurs sera créée comme si vous dessiniez un rectangle. Le premier clic définit un sommet du rectangle et le deuxième le sommet opposé.
  3. Appuyez sur Entrée, Échap ou cliquez avec le bouton droit pour terminer la commande.


Boîte de dialogue d'insertion d'une matrice d'arroseurs.



Remarque : la matrice d'arroseurs est créée sous forme de groupe d'objets. Vous pouvez cependant la dégrouper avec la commande Dégrouper Dans ce cas, vous obtiendrez des arroseurs individuels et les caractéristiques de la matrice seront perdues.

Points de contrôle

Les matrices d'arroseurs ont les mêmes points de contrôle que les courbes à partir desquelles elles ont été créées.

Elles disposent d'une flèche de contrôle du déplacement afin de déplacer l'objet dans son ensemble.

Pour activer les points de contrôle d'une matrice d'arroseurs, lancez la commande ActiverPoints (ou appuyez sur F10) et sélectionnez la matrice. Pour désactiver les points de contrôle, appuyez sur Échap ou lancez la commande DésactiverPoints.

Paramètres et options d'insertion

Les paramètres et options d'insertion des matrices d'arroseurs se trouvent dans différents onglets de plusieurs boîtes de dialogue.

  • Boîte de dialogue d'insertion d'un arroseur (uniquement disponible lors de l'insertion de nouvelles matrices d'arroseurs).
  • Section propriétés des matrices d'arroseurs, dans le panneau d'édition de Lands Design.
  • Boîte de dialogue des propriétés objet d'une matrice d'arroseurs.

Général

Attributs généraux de la matrice d'arroseurs : Nom, Couleur, Calque, Code et Mots clé.

Arroseur

  • Type : il existe quatre types d'arroseur :
    • Arroseur
    • Diffuseur
    • Turbine
    • Buse d'arrosage
  • Description : champ pour ajouter des notes sur les arroseurs.
  • Symbole : dessin 2D utilisé pour représenter les arroseurs dans la zone de la matrice. Cliquez sur le bouton Parcourir... pour charger un bloc 2D de la bibliothèque de Lands Design ou depuis un répertoire de votre ordinateur.
  • Rayon
  • Angle de balayage

Matrice

  • Distribution : les arroseurs peuvent être distribués dans un carré ou un triangle.
  • Distance entre les arroseurs : la distance est calculée sous forme de pourcentage du diamètre des arroseurs.

Options d'édition

Options d'édition des matrices d'arroseurs, disponibles dans la zone d'édition du panneau d'édition :

  • Actualiser
  • Copier les propriétés d'un autre objet.
  • Ajuster
    • Élévation
    • À l'élévation zéro
    • Au terrain
  • Ne pas ajuster au terrain
  • Matrice d'arroseurs
    • Ajouter un trou : commande pour soustraire une frontière de la zone d'une matrice d'arroseurs.

Affichage des matrices d'arroseurs :

Les matrices d'arroseurs peuvent être affichées simultanément en 2D et 3D dans le dessin.

Des blocs sont disponibles dans la bibliothèque de Lands Design pour représenter les arroseurs : C:\ProgramData\AsuniCAD\Lands\Rhinoceros (64-bit)\Blocks\Sprinklers\
