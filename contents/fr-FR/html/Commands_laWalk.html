Mode de déambulation

Clavier :
laDéambuler
Icône :
clic gauche
Barre d'outils : Outils de Lands
Menu :
Lands Design > Mode de déambulation
Panneau d'édition :
Ligne d'outils > Mode de déambulation

La commande Mode de déambulation vous permet de vous déplacer en marchant et en volant librement dans votre projet en utilisant le clavier et la souris.

Après avoir lancé la commande laDéambuler, cliquez sur la barre d'espace pour passer du Mode de déambulation au Mode de survol et vice versa :

  • Mode de déambulation : vous vous déplacez dans le projet en détectant l'élévation du terrain et les autres éléments.
  • Mode de survol : vous vous déplacez librement dans n'importe quelle direction et à n'importe quelle élévation dans le projet.

Contrôleurs de déambulation/survol

Souris

  • Pointeur : contrôle la direction où regarder.
  • Clic : avance.
  • Clic droit : recule.
  • Molette : augmente/diminue la vitesse de déambulation ou de survol.
  • Bouton central: définit la vitesse de déplacement par défaut.

Clavier

  • ESPACE : alterne entre le Mode de déambulation et le Mode de survol.
  • A : déplace l'observateur vers la gauche.
  • D : déplace l'observateur vers la droite.
  • W : avance.
  • S : recule.
  • + et - : contrôle la hauteur de l'observateur.
