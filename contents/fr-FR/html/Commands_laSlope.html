Lands Design > Documentation > Pente

La commande laPente dessine une ligne de d'annotation avec une pointe de flèche et un texte indiquant la pente du chemin sélectionné. La pente est calculée à partir de la longueur du chemin et de la différence en hauteur entre le point de départ et le point final du chemin.

Insérer une pente

Étapes :

  1. Sélectionnez une courbe du modèle pour définir la pente ou sélectionnez une option dans la ligne de commande.
    • Polyligne : la pente sera créée comme si vous dessiniez une polyligne.
    • Spline : la pente sera créée comme si vous dessiniez une spline.
    • Arc : la pente sera créée comme si vous dessiniez un arc. Le premier clic définit le point d'insertion. Le deuxième indique le point central de l'arc. Le troisième et le dernier déterminent les extrémités de l'arc.
  2. Appuyez sur Entrée, Échap ou cliquez avec le bouton droit pour terminer la commande.

Points de contrôle

Les pentes possèdent autant de points de contrôle que la courbe à partir de laquelle elles ont été créées.

Elles disposent d'une flèche de contrôle du déplacement afin de déplacer l'objet dans son ensemble.

Les points de contrôle des pentes sont automatiquement activés lorsqu'une pente est sélectionnée. Pour activer les points de contrôle sur une ou plusieurs pentes, sélectionnez-les et lancez la commande ActiverPoints (ou appuyez sur F10). Pour désactiver les points de contrôle, appuyez sur Échap ou lancez la commande DésactiverPoints.

Paramètres

Les paramètres des pentes se trouvent dans différents onglets de plusieurs boîtes de dialogue :

  • Section propriétés des pentes, dans le panneau d'édition de Lands Design.
  • Boîte de dialogue des propriétés objet d'une pente.

Général

Attributs généraux pour l'objet de cotation et d'origine des coordonnées : Nom, Couleur, Calque, Code et Mots clé.

Pente

Cet onglet affiche les paramètres de la pente :

  • Police : attributs et paramètres du texte.
  • Fin : options d'affichage de la pointe de flèche et du point de départ de la pente.

Remarque : Le nombre de chiffres après la virgule affichés dans le texte peut être défini et modifié dans le menu Outils > Options > Propriétés du document > Unités > Précision d'affichage.

Options d'édition

Options d'édition de la pente, disponibles dans la zone d'édition du panneau d'édition :

Options communes :

  • Actualiser
  • Copier les propriétés d'un autre objet
  • Copier sur une autre courbe
  • Inverser la courbe
  • Extraire une courbe
  • Ajuster
    • Élévation
    • À l'élévation zéro
    • Au terrain
  • Ne pas ajuster au terrain

Affichage des pentes :

Les pentes ont uniquement une représentation 2D qui est visible dans tous les modes d'affichage.
