Caractéristiques générales

Logiciel

Lands Design fonctionne sur Rhino 6 SR14 ou toute version postérieure.

Versions et mises à jour

Lands Design recherche automatiquement les mises à jour. Si une mise à jour est disponible, une icône apparaît dans la zone de notification de Windows

Langues

L'interface, les fichiers modèles et les barres d'outils de Lands Design sont disponibles en 6 langues : français, anglais, espagnol, italien, allemand et chinois (simplifié).

Une boîte de dialogue pour choisir la langue de Lands Design se trouve dans les Options de Lands Design ou dans la section Lands Design de la boîte de dialogue Options de Rhino.

Unités

Les systèmes d'unités métrique et impérial peuvent être utilisés dans Lands Design. Lorsque les unités du dessin sont modifiées, un message apparaît pour vous demander si vous voulez changer l'échelle du modèle en appliquant un facteur de conversion.

Calques

Vous pouvez configurer le calque par défaut utilisé par les commandes de création. Le panneau Calques se trouve dans les Options de Lands Design ou dans la section Lands Design de la boîte de dialogue Options de Rhino.

Lorsqu'une commande de création d'objets est lancée, le calque cible est utilisé comme calque actuel. Lorsque la commande se termine, le calque actif auparavant redevient le calque actuel sauf si l'utilisateur l'a changé pendant la commande.

Aide

Lands Design dispose d'options d'aide, qui sont répertoriées dans le menu principal de Lands Design.
Parmi celles-ci vous trouverez un fichier d'aide, des liens vers la section d'assistance et le site Internet de Lands Design ainsi qu'une boîte de dialogue contenant des informations sur la licence, le nom de l'utilisateur, l'entreprise et le numéro de série.
