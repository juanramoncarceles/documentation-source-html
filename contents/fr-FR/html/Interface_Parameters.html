Paramètres

Paramètres utilisés pour ajouter des informations aux objets de la géométrie. Ces objets peuvent être des éléments de Lands Design (plantes, clôtures, chemins, terrains, etc.) ou des éléments de Rhino (points, courbes, surfaces, polysurfaces, extrusions, maillages et blocs).

Les paramètres peuvent être créés et modifiés de deux manières différentes :

  • Par document
  • Par objet individuellement

Les paramètres existants sont visibles pour chaque entité d'objet dans le panneau Propriétés de Rhino , sous l'icône Paramètres , où il est possible d'ajouter les valeurs.

Les paramètres du document sont gérés dans la section Paramètres de la boîte de dialogue Propriétés du document de Rhino. Ces paramètres sont disponibles pour chaque objet de géométrie présent dans le modèle.

Les paramètres existants sont affichés dans une liste avec leur Catégorie et leur Type de valeur.


Section Paramètres des propriétés du document de Rhino

  • Ajouter : ouvre la boîte de dialogue des propriétés de paramètre afin de créer un nouveau paramètre.
  • Modifier : ouvre la boîte de dialogue des propriétés de paramètre afin de modifier un paramètre sélectionné dans la liste.
  • Supprimer : supprime un paramètre sélectionné dans la liste.

Les informations sur les paramètres personnalisés ajoutées par document aux éléments de Lands Design peuvent être listées avec les outils de liste de Lands Design.

Paramètres par objet

Les paramètres définis par objet sont ceux gérés pour un seul objet individuellement. Ils peuvent être créés, modifiés et supprimés à partir du panneau Propriétés de Rhino , avec l'icône des paramètres après avoir sélectionné un objet.

  • Ouvre la boîte de dialogue des propriétés de paramètre afin de créer un nouveau paramètre.
  • Ouvre la boîte de dialogue des propriétés de paramètre afin de modifier le paramètre. Vous devez sélectionner le nom du paramètre pour activer ce bouton.
  • Supprime un paramètre existant. Vous devez sélectionner le nom du paramètre pour activer ce bouton.

Remarque : Le nom et la description sont des paramètres natifs et ne peuvent pas être modifiés.

Boîte de dialogue des propriétés du paramètre


Boîte de dialogue des propriétés du paramètre

  • Nom : nom du paramètre.
  • Catégorie : section où sera visible le paramètre. Vous pouvez soit taper une nouvelle catégorie, soit sélectionner une des catégories existantes (Général, Affichage, Géométrie et Position), en plus des nouvelles catégories qui auraient pu déjà être créées.
  • Type de donnée : indique le format de la valeur. Sélectionnez une des options de la liste.
  • Description : champ pour ajouter des notes sur le paramètre.
