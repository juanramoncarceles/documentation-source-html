Lands Design > Documentation > Liste > Liste de mobilier urbain

La commande laListeBlocsBiblio insère un tableau 2D contenant les informations sur les blocs de mobilier urbain présents dans le modèle.

Les caractéristiques des listes de blocs de mobilier urbain sont les mêmes que celles des autres listes.

  • Options et paramètres d'insertion
  • Options d'édition
  • Affichage


Liste du mobilier urbain.

Champs des listes de mobilier urbain

Champs par défaut des listes de mobilier urbain, disponibles sous l'onglet Liste de mobilier urbain :

  • Icône de bloc : vous pouvez modifier la valeur d'échelle pour contrôler la taille de l'icône.
  • Index : identification du bloc (ID). Lands assigne une valeur ID aux blocs par défaut.
  • Identifiant
  • Nom
  • Quantité : nombre d'occurrences du bloc dans le modèle.


Onglet Liste de mobilier urbain dans le panneau d'édition.
