La commande laListeCanalisations insère un tableau 2D contenant les informations sur les canalisations présentes dans le modèle.

Les caractéristiques des listes de blocs de mobilier urbain sont les mêmes que celles des autres listes.

Liste des canalisations

Champs par défaut des listes de canalisations, disponibles sous l'onglet Liste de canalisations :

  • Icône de tuyau : petit dessin de canalisation affiché dans la liste.
  • Index : identification de la canalisation (ID). Lands assigne une valeur ID aux canalisations par défaut.
  • Section (diamètre) : cote de diamètre de la canalisation.
  • Pression : pression d'eau à l'intérieur des canalisations, exprimée en atm.
  • Matériau : description du matériau.
  • Description : texte affiché dans le champ de description de la canalisation.
  • Mètres linéaires : longueur de la canalisation, exprimée en mètres linéaires.
