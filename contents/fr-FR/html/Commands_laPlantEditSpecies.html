Modifier des espèces

Où puis-je trouver cette commande ?
Clavier :
laModifierEspèce
Menu :
Lands Design > Modifier > Modifier des espèces
Panneau d'édition :
Sélectionner un élément de Lands > Zone d'édition du panneau d'édition > Plante > Modifier des espèces

La commande laModifierEspèce sélectionne toutes les plantes de la même espèce dans le modèle et permet de modifier leurs caractéristiques communes (nom, hauteur, modes de représentation) dans la zone des propriétés d'objet du panneau d'édition de Lands.

Cette commande est disponible depuis la zone d'édition du panneau d'édition pour les objets suivants :

  • Plante
  • File de plantes
  • Forêt
  • Arbuste
  • Couvre-sol
