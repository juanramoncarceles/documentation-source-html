Insérer un escalier :

Lorsque vous lancez la commande laEscalier, la boîte de dialogue d'insertion d'un escalier apparaît. Cette boîte de dialogue présente les options de base de l'escalier dans deux onglets.

  • Escalier : paramètres de l'escalier.
  • Matériau : paramètres de matériau de l'escalier.

Étapes :

  1. Sélectionnez les paramètres de l'escalier. Cliquez sur Accepter pour fermer la boîte de dialogue.
  2. Sélectionnez une courbe du modèle pour définir le chemin de l'escalier ou sélectionnez une option dans la ligne de commande.
    • Polyligne : l'escalier sera créé comme si vous dessiniez une polyligne.
    • Spline : l'escalier sera créé comme si vous dessiniez une spline.
    • Cercle : l'escalier sera créé comme si vous dessiniez un cercle. Le premier clic définit le centre du cercle et le deuxième indique le rayon.
    • Arc : l'escalier sera créé comme si vous dessiniez un arc. Le premier clic définit le point d'insertion. Le deuxième indique le point central de l'arc. Le troisième et le dernier déterminent les extrémités de l'arc.
    • Rectangle : l'escalier sera créé comme si vous dessiniez un rectangle. Le premier clic définit un sommet du rectangle et le deuxième le sommet opposé.
  3. Appuyez sur Entrée, Échap ou cliquez avec le bouton droit pour terminer la commande.

Points de contrôle

Les escaliers ont les mêmes points de contrôle que les courbes à partir desquelles ils ont été créés.

Ils disposent d'une flèche de contrôle du déplacement afin de déplacer l'objet dans son ensemble.

Les points de contrôle des escaliers sont automatiquement activés lorsqu'un escalier est sélectionné. Pour activer les points de contrôle sur un ou plusieurs escaliers, sélectionnez-les et lancez la commande ActiverPoints (ou appuyez sur F10). Pour désactiver les points de contrôle, appuyez sur Échap ou lancez la commande DésactiverPoints.

Paramètres et options d'insertion

Les paramètres et options d'insertion des escaliers se trouvent dans différents onglets de plusieurs boîtes de dialogue.

  • Boîte de dialogue d'insertion d'un escalier (uniquement disponible lors de l'insertion de nouveaux escaliers).
  • Section propriétés des escaliers, dans le panneau d'édition de Lands Design.
  • Boîte de dialogue des propriétés objet d'un escalier.

Général

Attributs généraux de l'escalier : Nom, Couleur, Code, et Mots clé.

Escalier

  • Giron : détermine la distance longitudinale entre deux dalles consécutives.
  • Contremarche : détermine la distance verticale entre deux dalles consécutives.
  • Dalle : Les dalles sont les marches de l'escalier. Vous pouvez définir les dimensions suivantes des dalles :
    • Longueur : la profondeur de la dalle.
    • Hauteur : la hauteur de la dalle.
    • Largeur : la largeur est appliquée à tout l'escalier. Vous pouvez définir une largeur constante ou variable, du début à la fin de l'escalier.
    • Alignement : trois options sont disponibles pour aligner un escalier (gauche, milieu ou droite), en fonction de sa position définie à partir de la direction de sa courbe guide.

Matériau

Cet onglet permet de changer l'image de texture assignée à cet élément et de définir la taille, la réflectivité et la transparence de l'image.

Options d'édition

Options d'édition des escaliers, disponibles dans la zone d'édition du panneau d'édition :

  • Actualiser
  • Convertir en 2D. Affiche l'objet avec un affichage 2D.
  • Convertir en 3D. Affiche l'objet avec un affichage 3D.
  • Afficher en 2D et 3D : Active les représentations 2D et 3D de l'objet simultanément.
  • Copier les propriétés d'un autre objet
  • Copier sur une autre courbe
  • Inverser la courbe
  • Extraire une courbe
  • Ajuster
    • Élévation
    • À l'élévation zéro
    • Au terrain
  • Ne pas ajuster au terrain

Affichage des escaliers :
