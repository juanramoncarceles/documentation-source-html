Boîte de dialogue propriétés d’un objet

Clavier :
laPropriétés
Raccourci :
Double-clic sur l'objet à modifier

La boîte de dialogue des propriétés d’un objet est une boîte de dialogue flottante qui permet de modifier les propriétés des objets de Lands Design sélectionnés.

Pour ouvrir la boîte de dialogue des propriétés, lancez la commande laPropriétés et sélectionnez l'objet ou les objets à modifier. Appuyez sur Entrée ou cliquez avec le bouton droit pour terminer la sélection.


Boîte de dialogue des propriétés d'objet de Lands Design

La boîte de dialogue des propriétés d'objet est organisée en différents onglets suivant le type d'objet.

Onglets

  • Général :
    Tous les éléments de Lands Design ont cet onglet. Il détermine les attributs généraux de l'objet sélectionné, tels que le Nom, la Couleur, le Calque, le Code et les Mots-clé.
  • Rendu :
    Les plantes, les files de plantes, les forêts et arbustes ont cet onglet. Il définit comment les plantes seront affichées dans un fichier rendu, en fonction de la saison et de la densité de leur feuillage.
  • Plante :
    Les plantes, les files de plantes, les forêts et arbustes ont cet onglet. Il définit les propriétés de l'espèce de plantes.
  • File :
    Seules les files de plantes et les Files possèdent cet onglet. Il définit les paramètres des files et des files de plantes.
  • Couvre-sol :
    Seuls les couvre-sols présentent cet onglet. Il définit les paramètres des couvre-sols.
  • Mur :
    Seuls les murs possèdent cet onglet. Il définit les paramètres des murs.
  • Clôture :
    Seules les clôtures présentent cet onglet. Il définit les paramètres des clôtures.
  • Escalier :
    Seuls les escaliers présentent cet onglet. Il définit les paramètres des escaliers.
  • Chemin :
    Seuls les chemins présentent cet onglet. Il définit les paramètres des chemins.
  • Terrain :
    Seuls les terrains présentent cet onglet. Il définit les paramètres des terrains.
