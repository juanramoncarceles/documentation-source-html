Barre d'outils :
Menu :
Lands Design > Documentation > Liste > Liste des zones

La commande laListeZones insère un tableau 2D contenant les informations sur les zones présentes dans le modèle.

Les caractéristiques des listes de zones sont les mêmes que celles des autres listes.

  • Insérer une liste de zones
  • Options et paramètres d'insertion
  • Options d'édition
  • Affichage


Liste de zones.

Champs de la liste de zones

Champs par défaut des listes de zones, disponibles sous l'onglet Liste de zones :

  • Icône exemple : petit aperçu de chaque motif de hachures utilisé pour l'affichage 2D de la zone. Vous pouvez modifier la valeur d'échelle pour contrôler la taille de l'icône.
  • Index : identification de la zone (ID). Lands assigne une valeur ID aux zones par défaut.
  • Nom
  • Aire
