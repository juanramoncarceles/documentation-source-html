Insérer un chemin :

Lorsque vous lancez la commande laChemin, la boîte de dialogue d'insertion d'un chemin apparaît. Cette boîte de dialogue présente les options de base du chemin dans deux onglets.

  • Chemin : paramètres du chemin.
  • Matériau : paramètres de matériau pour le chemin.

Étapes :

  1. Sélectionnez le profil de section du chemin. Un profil rectangulaire (3 x 0,2 mètres) est déjà défini par défaut. Vous pouvez également sélectionner une courbe existante dans le modèle (option Sélectionner) ou la dessiner dans le modèle (option Choisir par point).
  2. Définissez les autres paramètres du chemin. Cliquez sur Accepter pour fermer la boîte de dialogue
  3. Sélectionnez une courbe du modèle pour définir le chemin ou sélectionnez une option dans la ligne de commande.
    • Polyligne : le chemin sera créé comme si vous dessiniez une polyligne.
    • Spline : le chemin sera créé comme si vous dessiniez une spline.
    • Cercle : le chemin sera créé comme si vous dessiniez un cercle. Le premier clic définit le centre du cercle et le deuxième indique le rayon.
    • Arc : le chemin sera créé comme si vous dessiniez un arc. Le premier clic définit le point d'insertion. Le deuxième indique le point central de l'arc. Le troisième et le dernier déterminent les extrémités de l'arc.
    • Rectangle : le chemin sera créé comme si vous dessiniez un rectangle. Le premier clic définit un sommet du rectangle et le deuxième le sommet opposé.
  4. Appuyez sur Entrée, Échap ou cliquez avec le bouton droit pour terminer la commande.

Remarque : vérifiez que la courbe sélectionnée comme profil de section du chemin est alignée avec les axes XY.

Points de contrôle

Les chemins ont trois types de points de contrôle :

  1. Points de contrôle de l'axe du chemin : ce sont les mêmes points de contrôle que ceux des courbes à partir desquelles ils ont été créés.
  2. Flèche de contrôle du déplacement : flèche se trouvant au début du chemin, qui est utilisée pour déplacer tout l'objet.
  3. Points de contrôle du profil de section : ces points de contrôle sont utilisés pour modifier le profil de section du chemin. Ce sont les mêmes points de contrôle que ceux des courbes à partir desquelles le profil de section est défini. Il se trouvent au début du chemin.

Les points de contrôle des chemins sont automatiquement activés lorsqu'un chemin est sélectionné. Pour activer les points de contrôle sur un ou plusieurs chemins, sélectionnez-les et lancez la commande ActiverPoints (ou appuyez sur F10). Pour désactiver les points de contrôle, appuyez sur Échap ou lancez la commande DésactiverPoints.

Options et paramètres d'insertion

Les paramètres et options d'insertion des chemins se trouvent dans différents onglets de plusieurs boîtes de dialogue.

  • Boîte de dialogue d'insertion d'un chemin (uniquement disponible lors de l'insertion de nouveaux chemins).
  • Section propriétés des chemins, dans le panneau d'édition de Lands Design.
  • Boîte de dialogue des propriétés objet d'un chemin.

Général

Attributs généraux pour un chemin : Nom, Couleur, Calque, Code et Mots clé.

Trajectoire

  • Profil de la section : courbe utilisée pour définir le profil de section du chemin.
  • Aspect lisse : précision avec laquelle le chemin est subdivisé le long de la courbe de l'axe afin de s'adapter à la forme. Ce paramètre n'est important que lorsque le chemin a été créé à partir d'une courbe présentant une courbure.
  • Alignement : marque la position du profil de section du chemin à partir duquel le chemin est extrudé le long de la courbe de l'axe. Il existe neuf alignements possibles. L'option Aucun conserve l'alignement du chemin le long de l'axe, même si vous modifiez le profil de section.

Matériau

Cet onglet permet de changer l'image de texture assignée à cet élément et de définir la taille et la réflectivité de l'image.

Options d'édition

Options d'édition du chemin, disponibles dans la zone d'édition du panneau d'édition :

  • Actualiser
  • Convertir en 2D. Affiche l'objet avec une représentation 2D.
  • Convertir en 3D. Affiche l'objet avec une représentation 3D.
  • Afficher en 2D et 3D : Active les représentations 2D et 3D de l'objet simultanément.
  • Copier les propriétés d'un autre objet
  • Copier sur une autre courbe
  • Inverser la courbe
  • Extraire une courbe
  • Ajuster
    • Élévation
    • À l'élévation zéro
    • Au terrain
  • Ne pas ajuster au terrain
  • Trajectoire
    • Modifier le profil de la section : cette commande place un plan de construction aligné avec le profil de section du chemin au début de l'objet. Il est ainsi plus facile de modifier le profil du chemin avec ses points de contrôle. Lancez la commande PlanC (et sélectionnez RepèreGénéral > Dessus) dans la ligne de commande pour rétablir la position du plan de construction.

Affichage des chemins :

Les chemins peuvent être affichés simultanément en 2D et 3D dans le dessin.
