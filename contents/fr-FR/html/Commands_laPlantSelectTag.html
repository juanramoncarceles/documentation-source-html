Identifier une étiquette

Clavier :
laPlanteSélectionnerEtiquette
Icône :
clic gauche
Barre d'outils :
Menu :
Lands Design > Documentation > Étiquettes de plantes > Identification de l'étiquette

La commande laPlanteSélectionnerEtiquette vous aide à identifier les étiquettes d'un ensemble de plantes sélectionnées.

Étapes :

  1. Sélectionnez la ou les plantes dont vous voulez identifier l'étiquette. Cliquez avec le bouton droit ou appuyez sur Entrée une fois terminé.

L'étiquette sera affichée en surbrillance.
