Lier des étiquettes de plantes

Où puis-je trouver cette commande ?
Clavier :
laPlanteÉtiquetteLier
Icône :
clic gauche
Barre d'outils : Étiquette de plantes de Lands
Menu :
Lands Design > Documentation > Étiquette de plante > Lier des étiquettes

La commande laPlanteÉtiquetteLier insère une étiquette 2D contenant les informations sur un groupe de plantes sélectionnées dans le modèle (quantité de spécimens et ID des espèces de plantes). Une polyligne relie les plantes sélectionnées avec l'étiquette.

Une liste de plantes est nécessaire pour insérer des étiquettes de plantes.

Étapes :

  1. Sélectionnez les espèces de plantes que vous voulez étiqueter, en suivant l'ordre dans lequel vous voulez les relier.
  2. Indiquez le point d'insertion où placer l'étiquette de plante. Vous devrez insérer autant d'étiquettes que d'espèces sélectionnées.

Les informations contenues dans les étiquettes contiennent deux valeurs :

  • Valeur de gauche : quantité d'unités appartenant à la même espèce de plantes.
  • Valeur de droite : code d'identification de l'espèce de plante (ID) conformément à la liste de plantes. Cet ID peut être remplacé par les initiales du nom de l'espèce de plantes en cochant l'option Utiliser les initiales dans les propriétés de la liste de plantes.


Étiquettes de plantes liées.

Paramètres des étiquettes de plantes

Les paramètres des étiquettes de plantes se trouvent dans différents onglets de plusieurs boîtes de dialogue :

  • Section propriétés des étiquettes de plantes, dans le panneau d'édition de Lands Design.
  • Boîte de dialogue des propriétés objet d'une étiquette de plante.

Général, Étiquette et Ligne de repère

Paramètres des étiquettes de plantes qui définissent la forme, le format et les attributs de l'étiquette.

Points de contrôle

Les étiquettes de plantes ont un seul point de contrôle se trouvant au centre du contour de l'étiquette.


Options d'édition

Options d'édition des étiquettes de plantes, disponibles dans la zone d'édition du panneau d'édition :

  • Actualiser
  • Copier les propriétés d'un autre objet
  • Ajuster
    • Élévation
    • À l'élévation zéro
    • Au terrain
  • Ne pas ajuster au terrain
  • Étiquettes des plantes
    • Identifier des plantes
    • Ajouter ou supprimer des plantes dans une étiquette
