Lands Design > Documentation > Liste > Liste des arroseurs

La commande laListeArroseurs insère un tableau 2D contenant les informations sur les arroseurs présents dans le modèle.

Les caractéristiques des listes d'arroseurs sont les mêmes que celles des autres listes.

  • Insère une liste d'arroseurs
  • Options et paramètres d'insertion
  • Options d'édition
  • Affichage


Liste des arroseurs.

Champs de la liste des arroseurs

Champs par défaut des listes d'arroseurs, disponibles sous l'onglet Liste d'arroseurs :

  • Icône de l'arroseur : le symbole de l'arroseur. Vous pouvez modifier la valeur d'échelle pour contrôler la taille de l'icône.
  • Index : identification de l'arroseur (ID). Lands assigne une valeur ID aux arroseurs par défaut.
  • Type : type d'arroseur.
  • Rayon
  • Angle de balayage
  • Description : texte affiché dans le champ de description de l'arroseur.
  • Quantité : nombre d'arroseurs dans le modèle. La liste affichera les arroseurs individuellement s'ils ont des propriétés différentes (Type, Angle de balayage, Rayon, etc.) et si ces propriétés sont activées dans la liste.
