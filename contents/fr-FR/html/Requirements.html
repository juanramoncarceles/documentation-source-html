Configuration requise

Logiciel

Lands Design fonctionne sur Rhino 6 SR14.

Matériel

La configuration minimale du système est la même que celle de Rhino 6.

Installation

Lands Design possède un assistant d'installation intuitif qui permet d'automatiser le processus.

L'installation est réalisée par défaut dans : C:\Program Files\Lands Design (Rhino 6).

Une fois installé, cliquez sur l'icône d'accès direct à Lands Design pour lancer le programme. Une boîte de dialogue vous demandera le système d'unités que vous voulez utiliser :

  • Métrique (International) - (Lands Design.3dm)
  • Anglo-saxon (International) - (Lands Design-Anglo-saxon .3dm)


Liste de modèles de Lands Design

Si vous voulez commencer un nouveau document, ouvrez le menu Fichier de Rhino > Nouveau et ouvrez le dossier Lands Design dans la boîte de dialogue des fichiers modèles de Rhino et sélectionnez un modèle de Lands Design.

Les fichiers modèles de Lands Design sont enregistrés dans le répertoire suivant :
C:\Users\"Nom d'utilisateur"\AppData\Roaming\McNeel\Rhinoceros\"RhinoVersion"\Localization\fr-FR\Template Files\Lands Design
(Ce répertoire correspond au dossier des fichiers modèles en français. Remplacez "fr-fr" par un autre code de langue pour trouver les autres répertoires où sont stockés les fichiers modèles de Lands Design dans d'autres langues).
