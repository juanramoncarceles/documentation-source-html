Copier les propriétés d'un autre objet

Où puis-je trouver cette commande ?
Clavier :
laCopierDepuis
Menu :
Lands Design > Modifier > Copier les propriétés d'un autre objet
Panneau d'édition :
Sélectionner un élément de Lands > Zone d'édition du panneau d'édition > Copier les propriétés d'un autre objet

La commande laCopierDepuis permet de changer les propriétés d'un objet de Lands afin qu'elles coïncident avec celles d'un autre objet de Lands.

Cette commande est disponible depuis la zone d'édition du panneau d'édition pour les objets suivants :

  • File de plantes
  • Forêt
  • Arbuste
  • Mur
  • Clôture
  • File
  • Trajectoire
  • Escalier
  • Zone

Étapes :

  1. Sélectionnez l'objet à modifier.
  2. Sélectionnez l'objet dont vous voulez copier les propriétés.
