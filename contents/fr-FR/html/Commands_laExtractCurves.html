Extraire des courbes

Où puis-je trouver cette commande ?
Clavier :
laExtraireCourbes
Menu :
Lands Design > Modifier > Extraire des courbes
Panneau d'édition :
Sélectionner un élément de Lands > Zone d'édition du panneau d'édition > Extraire des courbes

La commande laExtraireCourbes permet d'extraire les courbes utilisées pour générer les éléments de Lands.

Cette commande est disponible depuis la zone d'édition du panneau d'édition pour les objets suivants :

  • File de plantes
  • Forêt
  • Arbuste
  • Mur
  • Clôture
  • File
  • Trajectoire
  • Escalier
  • Zone

Étapes :

  1. Sélectionnez l'objet dont vous voulez extraire les courbes.
