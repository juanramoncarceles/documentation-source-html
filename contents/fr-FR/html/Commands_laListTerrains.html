Lands Design > Documentation > Liste > Liste des arroseurs

La commande laListeTerrains insère un tableau 2D contenant les informations sur les terrains présents dans le modèle ainsi que les opérations réalisées dessus.

Les caractéristiques des listes de terrains sont les mêmes que celles des autres listes.

Champs de la liste de terrains

Champs par défaut des listes de terrains, disponibles sous l'onglet Liste de terrains :

  • Nom : nom donné au terrain.
  • Index : identification du terrain (ID).
  • Aire: aire à plat de l'opération appliquée sur le terrain.
  • Volume de remplissage : quantité de terrassement ajouté au terrain.
  • Volume de creusement : quantité de terrassement soustrait dans le terrain.
