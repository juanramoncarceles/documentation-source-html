Insérer un arroseur

Lorsque vous lancez la commande laArroseur, la boîte de dialogue d'insertion d'un arroseur apparaît. Cette boîte de dialogue présente les options de base dans un onglet unique.

  • Arroseur : paramètres de l'arroseur.

Étapes :

  1. Définissez les paramètres de l'arroseur. Cliquez sur Accepter pour fermer la boîte de dialogue.
  2. Sélectionnez le point d'insertion de l'arroseur en cliquant sur le point désiré dans le modèle.
  3. Ensuite, indiquez l'orientation de l'arroseur manuellement avec le curseur. Cliquez à nouveau pour insérer un autre arroseur ou...
  4. Appuyez sur Entrée, Échap ou cliquez avec le bouton droit pour terminer la commande.


Boîte de dialogue d'insertion d'un arroseur.



Points de contrôle

L'objet représentant l'arroseur possède un seul point de contrôle en son centre.

Pour activer le point de contrôle d'un arroseur, lancez la commande ActiverPoints (ou appuyez sur F10) et sélectionnez l'arroseur. Pour désactiver le point de contrôle, appuyez sur Échap ou lancez la commande DésactiverPoints.


Point de contrôle activé sur un arroseur

Paramètres et options d'insertion

Les paramètres et options d'insertion des arroseurs se trouvent dans différents onglets de plusieurs boîtes de dialogue.

  • Boîte de dialogue d'insertion d'un arroseur (uniquement disponible lors de l'insertion de nouveaux arroseurs).
  • Section propriétés des arroseurs, dans le panneau d'édition de Lands Design.
  • Boîte de dialogue des propriétés objet d'un arroseur.

Général

Attributs généraux de l'arroseur : Nom, Couleur, Calque, Code et Mots clé.

Arroseur

  • Type : il existe quatre types d'arroseur :
    • Arroseurs : les arroseurs ont une portée de 6 m, ce qui signifie qu'ils peuvent envoyer de l'eau jusqu'à 6 m devant eux, en fonction du niveau de pression ou du type de buse utilisé.
    • Diffuseurs : les diffuseurs sont similaires aux arroseurs mais sont plus petits. Ils ont une portée de 2 à 5 m, en fonction de la pression et de la buse. Ils sont utilisés pour arroser des zones plus étroites.
    • Turbines : conviennent aux zones de taille moyenne longues et étroites.
    • Buse d'arrosage : point d'eau vissé dans le sol
  • Description : champ pour ajouter des notes sur les arroseurs.
  • Symbole : dessin 2D utilisé pour représenter les arroseurs dans la zone de la matrice. Cliquez sur le bouton Parcourir... pour charger un bloc 2D de la bibliothèque de Lands Design ou depuis un répertoire de votre ordinateur.
  • Rayon
  • Angle de balayage

Options d'édition

Options d'édition des arroseurs, disponibles dans la zone d'édition du panneau d'édition :

  • Actualiser
  • Copier les propriétés d'un autre objet
  • Ajuster
    • Élévation
    • À l'élévation zéro
    • Au terrain
  • Ne pas ajuster au terrain
  • Arroseur
    • Réorienter : changer la direction de l'arroseur

Affichage des arroseurs :

Les arroseurs sont représentés sous forme de références de blocs 2D. Des blocs sont disponibles dans la bibliothèque de Lands Design : C:\ProgramData\AsuniCAD\Lands\Rhinoceros (64-bit)\Blocks\Sprinklers\.

Pour créer un affichage personnalisé pour la représentation des arroseurs, créez le dessin dans un document vide en plaçant sont point central à l'origine des coordonnées. Enregistrez le fichier.
