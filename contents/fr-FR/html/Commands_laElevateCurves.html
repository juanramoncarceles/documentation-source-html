Élever des courbes

Où puis-je trouver cette commande ?
Clavier :
laEleverCourbes
Icône :
clic gauche
Barre d'outils : Terrain de Lands :
Menu :
Lands Design > Modélisation de terrain > Élever des courbes

Cette commande élève des courbes sur une certaine distance afin de les utiliser pour construire un terrain.

Lorsque vous lancez la commande laEleverCourbes, la boîte de dialogue Élever des courbes apparaît.

Étapes :

  1. Définissez les paramètres d'élévation des courbes dans la boîte de dialogue :
    • Cote de départ : Élévation de la première courbe.
    • Incrément. Distance entre les courbes de niveau à élever.
  2. Cliquez sur le bouton Dessiner une ligne et dessinez une ligne dans le modèle qui coupe les courbes de niveau à élever.
  3. Cliquez dans la case Inverser pour inverser l'ordre dans lequel les courbes sont élevées.
  4. Cliquez sur le bouton Appliquer et fermez la boîte de dialogue.
