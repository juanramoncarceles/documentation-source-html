La commande laListePlantes insère un tableau 2D contenant les informations sur les espèces de plantes présentes dans le modèle.

Les caractéristiques des listes de plantes sont les mêmes que celles des autres listes.

Champs de la liste des plantes

Champs par défaut des listes de plantes, disponibles sous l'onglet Liste de plantes :

  • Icône de l'espèce : dessin de la cime de l'espèce de plantes. Vous pouvez modifier la valeur d'échelle pour contrôler la taille de l'icône.
  • Index : identification de l'espèce (ID). Vous pouvez sélectionner une valeur numérique différente pour chaque espèce, calculée par Lands Design, ou utiliser les initiales des espèces de plantes.
  • Initiales : initiales de l'espèce de plantes.
  • Nom scientifique : nom scientifique de l'espèce.
  • Nom commun : nom commun de l'espèce.
  • Mesure : calibre (diamètre) des plantes de cette espèce insérées dans le dessin. Lorsque ce champ est affiché, la liste classe et compte séparément les spécimens de la même espèce mais de diamètres différents.
  • Présentation : racine nue, motte ou godet.
  • Densité : nombre de plantes par mètre carré qui font partie de la même structure de plante. La densité aura une valeur d'unité pour les spécimens insérés individuellement et une valeur réelle pour les arbustes et les couvre-sols. Lorsque ce champ est affiché, la liste classe et compte séparément les spécimens de la même espèce mais de densités différentes.
  • Quantité : quantité de plantes de cette espèce insérées dans le dessin. La quantité est la somme de toutes les plantes de cette espèce nécessaires pour le plan d'implantation. La quantité comprend les plantes nécessaires pour créer les arbustes et les couvre-sols du plan.
