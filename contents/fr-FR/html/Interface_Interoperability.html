Interopérabilité de Lands Design avec les commandes et l'interface de Rhino

Commandes de Rhino

Étant donné que les objets de Lands Design sont en réalité des blocs de Rhino, toutes les commandes de Rhino fonctionnant avec les blocs fonctionneront également par défaut avec les objets de Lands Design. D'autres commandes ont été adaptées, comme par exemple les commandes qui dupliquent des objets (telles que Copier, Matrice et Symétrie) ainsi que certaines commandes d'édition de Windows (telles que Couper, Copier et Coller : CTRL+X, CTRL+C, CTRL+V respectivement).
Lorsque des commandes de Rhino qui ne sont pas compatibles avec Lands Design sont lancées, les objets de Lands Design ne peuvent pas être sélectionnés.

La commande Décomposer supprime les propriétés des objets de Lands Design et les convertit ainsi en occurrences de bloc, maillages ou polysurfaces. Dans ce cas, vous pouvez les modifier comme toutes les autres polysurfaces de Rhino.
Contrairement aux autres objets de Lands Design, les objets de végétation (plantes, couvre-sols et arbustes) ne peuvent pas être décomposés dans Lands Design.

Comportement des commandes de Rhino

  • Répéter les dernières commandes utilisées :
    Comme pour les commandes de Rhino, vous pouvez répéter la plupart des commandes de Lands Design ainsi :
    • Pour répéter la dernière commande, appuyez sur Entrée, sur la barre d'espace ou cliquez avec le bouton de droite lorsque aucune commande n'est en cours.
    • Pour répéter une commande utilisée récemment, cliquez avec le bouton de droite dans la ligne de commande.
  • Sélectionner des objets :
    Comme pour les commandes de Rhino, la plupart des opérations de Lands Design vous demandent de sélectionner un ou plusieurs objets. Les objets de Lands Design sont sélectionnés comme les objets de Rhino :
    • Pour sélectionner un seul objet, cliquez dessus.
    • Pour sélectionner plusieurs objets, laissez enfoncée la touche Maj tout en cliquant sur les objets, ou utilisez une fenêtre de sélection.
  • Aides à la modélisation et accrochages aux objets :
    Les aides à la modélisation (magnétisme de la grille, mode Ortho et mode planéité) et les accrochages aux objets (fin, proche, point, mi, cen, int, perp, sommet etc.) peuvent être utilisés en appliquant les mêmes méthodes que pour les objets de Rhino. La plupart des objets de Lands Design étant des maillages, l'accrochage Sommet sera souvent utilisé.

Points de contrôle

Clavier :
ActiverPoints / DésactiverPoints
Icône :
Désactiver les points de contrôle : Bouton de droite
Menu :
Édition > Points de contrôle > Activer les points de contrôle / Désactiver les points de contrôle
Raccourci :
Activer les points de contrôle (F10), Désactiver les points de contrôle (F11)

Tous les objets de Lands Design peuvent être édités en activant leurs points de contrôle. Pour ce faire, vous pouvez utiliser les mêmes commandes que dans Rhino.
