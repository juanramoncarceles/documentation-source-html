Copier sur une autre courbe

Où puis-je trouver cette commande ?
Clavier :
laCopierSurCourbe
Menu :
Lands Design > Modifier > Copier sur une autre courbe
Panneau d'édition :
Sélectionner un élément de Lands > Zone d'édition du panneau d'édition > Copier sur une autre courbe

La commande laCopierSurCourbe crée de nouveaux éléments de Lands en sélectionnant des éléments existants et en appliquant leurs propriétés sur de nouvelles courbes.

Cette commande est disponible depuis la zone d'édition du panneau d'édition pour les objets suivants :

  • File de plantes
  • Forêt
  • Mur
  • Clôture
  • File
  • Trajectoire
  • Escalier
  • Zone

Étapes :

  1. Sélectionnez l'élément de Lands dont vous voulez appliquer les propriétés sur un nouvel objet.
  2. Sélectionnez une courbe du modèle pour définir le chemin ou la frontière du nouvel objet. Ou sélectionnez une option de la ligne de commande pour dessiner le nouvel objet en fonction de différentes courbes :
    • Polyligne : l'objet sera créé comme si vous dessiniez une polyligne.
    • Spline : l'objet sera créé comme si vous dessiniez une spline.
    • Cercle : l'objet sera créé comme si vous dessiniez un cercle. Le premier clic définit le centre du cercle et le deuxième indique le rayon.
    • Arc : l'objet sera créé comme si vous dessiniez un arc. Le premier clic définit le point d'insertion. Le deuxième indique le point central de l'arc. Le troisième et le dernier déterminent les extrémités de l'arc.
    • Rectangle : l'objet sera créé comme si vous dessiniez un rectangle. Le premier clic définit un sommet du rectangle et le deuxième le sommet opposé.
