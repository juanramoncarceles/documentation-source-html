Boîtes de dialogue d'insertion

Les boîtes de dialogue d'insertion affichent les options des objets qui vont être insérés. Lorsque les options d'insertion sont définies, cliquez sur Accepter pour fermer la boîte de dialogue et commencer à insérer les objets dans le modèle.


Boîte de dialogue d'insertion d'une Plante.

La boîte de dialogue est organisée en onglets, qui affichent les différentes options d'insertion.
