Importer un fichier MNT

Où puis-je trouver cette commande ?
Clavier :
laImporterMNT
Menu :
Lands Design > Modélisation de terrain > Importer un fichier MNT

Cette commande permet de générer un terrain à partir d'un fichier de modèle numérique de terrain.

Un fichier de modèle numériques de terrain (MNT) est une simple grille de valeurs d'élévation espacées régulièrement.

Lorsque vous lancez la commande laImporterMNT, la boîte de dialogue Importer les données d'élévation de la Terre apparaît.


Boîte de dialogue Importer les données d'élévation de la Terre.

Cette boîte de dialogue est divisée en deux parties. Une zone de navigation dans l'image à partir du fichier du MNT et les paramètres d'importation.

Étapes :

  1. Cliquez sur le bouton Fichier de MNT et parcourez les fichiers de votre ordinateur pour trouver un fichier de MNT au format compatible (*.tif, *.tiff, *.png, *.txt or *.asc ). Le fichier sera importé sous forme d'image en noir et blanc dont les pixels représentent différentes valeurs d'élévation.
  2. Dessinez un rectangle sur l'image pour définir la zone que vous voulez importer.
  3. Définissez les paramètres d'importation :
    • Courbes : importe les courbes de niveau scannées à partir de l'image. Vous pouvez choisir le calque et la couleur utilisés pour leur création. Ces courbes peuvent être utilisées plus tard pour générer le terrain avec la commande laTerrain.
    • Maillage : un maillage étiqueté comme terrain, contenant les points d'élévation, sera créé. Vous pouvez choisir le calque et la couleur utilisés pour sa création.
    • Terrain : un terrain sera automatiquement créé avec les données d'élévation scannées.
  4. Cliquez sur le bouton Importer pour importer la zone sélectionnée.
