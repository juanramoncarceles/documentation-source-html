Les terrains de Lands Design sont des maillages topographiques créés en élevant une surface à partir de courbes 3D, de lignes de niveau ou de nuages de points.

Insérer un terrain :

Lorsque vous lancez la commande laTerrain, la boîte de dialogue d'insertion d'un terrain apparaît. Cette boîte de dialogue présente les options de base du terrain dans deux onglets.

  • Terrain : paramètres du terrain.
  • Matériau : paramètres de matériau pour le terrain.

Étapes :

  1. Définissez les autres paramètres d'insertion du terrain et cliquez sur Accepter pour fermer la boîte de dialogue.
  2. Sélectionnez les données de départ du terrain, qui peuvent être des courbes ou des points. Appuyez sur Entrée pour terminer la commande.

Le terrain sera créé dans le modèle.


Boîte de dialogue d'insertion d'un terrain.



Les terrains peuvent être générées à partir de différentes données de départ.


Points de contrôle

Les terrains ont les mêmes points de contrôle que les courbes à partir desquelles ils ont été créés et que les courbes utilisées pour réaliser des opérations dessus (telles que celles utilisées avec la commande Terrain > Ajouter un trou, Terrain > Ajouter déblai et remblai, etc.). De plus, une flèche de contrôle du déplacement apparaît près de chacune de ces courbes afin de pouvoir la déplacer.

Les points de contrôle des terrains sont automatiquement activés lorsqu'un terrain est sélectionné. Pour activer les points de contrôle sur un ou plusieurs terrains, sélectionnez-les et lancez la commande ActiverPoints (ou appuyez sur F10). Pour désactiver les points de contrôle, appuyez sur Échap ou lancez la commande DésactiverPoints.

Les points de contrôle de la géométrie utilisée pour générer le terrain peuvent être désactivés individuellement à partir de l'onglet Données de départ dans la zone des propriétés du panneau d'édition.


Points de contrôle activés sur des terrains.

Paramètres et options d'insertion

Les paramètres et options d'insertion des terrains se trouvent dans différents onglets de plusieurs boîtes de dialogue.

  • Boîte de dialogue d'insertion d'un terrain (uniquement disponible lors de l'insertion de nouveaux terrains).
  • Section propriétés des terrains, dans le panneau d'édition de Lands Design.
  • Boîte de dialogue des propriétés objet d'un terrain .

Général

Attributs généraux du terrain : Nom, Couleur, Code, et Mots clé.

Terrain

Génération : Lands Design utilise la Triangulation de Delaunay pour générer les terrains. Il s'agit d'un algorithme très courant de génération de terrains, basé sur une triangulation optimale à partir des points dont l'élévation est connue.

Les terrains peuvent être créés en utilisant directement les faces de triangulation ou en ajustant une grille sur la forme obtenue après la triangulation, afin d'obtenir une surface lisse.

Grille rectangulaire adaptée à la triangulation : Lorsque cette option est activée, vous pouvez contrôler les paramètres suivants :

  • Taille de cellule
  • Rotation
  • Lisser
  • Simplifier
  • Utiliser le modèle de grille pour modéliser les trous ainsi que les déblais et remblais

Hauteur de la base :

Données de départ

Cet onglet est divisé en deux parties. La partie gauche affiche la liste des données de départ (courbes et points) utilisées pour générer le terrain et pour les opérations appliquées ensuite pour le modifier. La partie droite affiche les paramètres disponibles pour chaque sélection.

Lorsqu'une courbe ou opération est modifiée ou supprimée de la liste, le terrain est automatiquement recalculé.

Courbes de niveau

Équidistance (des courbes de niveau) : deux options sont disponibles pour générer les courbes de niveau d'un terrain pour sa représentation 2D :

  • Nombre de courbes de niveau : divise la hauteur du terrain en un nombre fixe de courbes de niveau.
  • Distance entre les courbes de niveau : crée des courbes de niveau à une certaine distance les unes des autres.

Apparence : paramètres d'affichage 2D pour les courbes de niveau du terrain. Vous pouvez définir des courbes de niveau indexées tous les "X" courbes de niveau, ajouter du texte et leur attribuer des propriétés de ligne différentes.

Options d'édition

Options d'édition des terrains, disponibles dans la zone d'édition du panneau d'édition :

  • Actualiser
  • Convertir en 2D. Affiche l'objet avec une représentation 2D.
  • Convertir en 3D. Affiche l'objet avec une représentation 3D.
  • Afficher en 2D et 3D : Active les représentations 2D et 3D de l'objet simultanément.
  • Copier les propriétés d'un autre objet
  • Ajuster
    • Élévation
    • À l'élévation zéro
  • Terrain
    • Sélectionner une frontière
    • Supprimer une frontière. (Cette commande apparaît uniquement lorsqu'une nouvelle frontière a été ajoutée au terrain)
    • Ajouter une courbe de niveau
    • Ajouter un trou
    • Ajouter déblai et remblai
    • Ajouter un chemin
    • Diviser
    • Volume de terrassement

Toutes les commandes pour générer, éditer et réaliser des opérations sur des terrains sont regroupées dans la barre d'outils Terrain de Lands.


Barre d'outils Terrain de Lands Design

Affichage graphique du terrain :

Lands Design offre simultanément plusieurs affichages 2D et 3D des terrains dans le dessin. Le nombre de courbes de niveau dans l'affichage 2D dépend des paramètres des lignes de niveau.
