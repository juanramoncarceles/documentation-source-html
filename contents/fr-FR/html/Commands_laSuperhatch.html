Remplissage avancé

Où puis-je trouver cette commande ?
Clavier :
laRemplissageAvancé
Menu :
Lands Design > Documentation > Remplissage avancé

La commande de remplissage avancé crée une surface texturée à partir de la répétition d'une image.

  • Insérer un remplissage avancé
  • Points de contrôle d'un remplissage avancé
  • Paramètres et options d'insertion
  • Affichage

Étapes :

  1. Lancez la commande laRemplissageAvancé. Sélectionnez une des deux options dans la ligne de commande pour insérer le remplissage avancé :
    • Indiquer un point intérieur : cliquez à l'intérieur d'une frontière fermée de courbes 2D.
    • Courbe frontière: sélectionnez une courbe fermée et plane.
  2. Après la première étape, la boîte de dialogue d'insertion d'un remplissage avancé apparaît. Cette boîte de dialogue présente les options de base du remplissage avancé dans un onglet unique :
    • Remplissage avancé : paramètres du remplissage avancé.
    Choisissez les options d'insertion de base. Appuyez sur Entrée, Échap ou cliquez avec le bouton droit pour fermer la boîte de dialogue.


Remplissage avancé.



Points de contrôle

Le remplissage avancé possède les mêmes points de points de contrôle que les courbes utilisées pour le générer ou que la courbe de frontière générée à partir des limites autour du point sélectionné lors de l'insertion.

Il dispose également d'une flèche de contrôle afin de déplacer l'objet dans son ensemble.

Pour activer les points de contrôle d'un remplissage avancé, lancez la commande ActiverPoints (ou appuyez sur F10) et sélectionnez le remplissage avancé. Pour désactiver les points de contrôle, appuyez sur Échap ou lancez la commande DésactiverPoints.

Paramètres et options d'insertion

Les paramètres et options d'insertion des remplissages avancés se trouvent dans différents onglets de plusieurs boîtes de dialogue.

  • Boîte de dialogue d'insertion d'un remplissage avancé (uniquement disponible lors de l'insertion de nouveaux remplissages avancés).
  • Section propriétés des remplissages avancés, dans le panneau d'édition de Lands Design.
  • Boîte de dialogue des propriétés objet d'un remplissage avancé.

Général

Attributs généraux du remplissage avancé : Nom, Couleur, Calque, Code et Mots clé.

Remplissage avancé

Cliquez sur le bouton Parcourir... pour sélectionner une image dans la bibliothèque de textures de Lands Design ou depuis votre ordinateur. Modifiez les valeurs d'échelle et de rotation.

Affichage des remplissages avancés :

Le remplissage avancé est représenté sous forme de surface texturée visible dans tous les modes d'affichage.
